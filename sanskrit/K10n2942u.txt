 sarva eva bālapṛthagjanāḥ kāmadhātau manuṣyapretatiryaṅ narakeyā yo'ya kāmasevanāt kāmadhātur bhavati 
 ārūpyās tu samāpattisamālambanāhārāḥ 
 tadevam aya dvābhyā baddho lokaḥ 
so virāgān na mucyate sarvasayojanānuśa yabhandhenebhyaḥ 
  asau anyena prakāreṇa cakṣur yathābhūta paśyati cakṣur vijñeyāṇy api rūpāṇi 
yad api tac cakṣusasparśanād utpadyate sukhasthānīya sukhālambanam asukhavipāka karma
tad api yathābhūta prajānāti 
eva daurmanasyasthānīya cakṣurvi jñeya yad api tac cakṣuḥsasparśajam utpadyate duḥkhā lambana sukhavipāka karma
tad api yathābhūta prajānāti 
katarat tac cakṣurvijñeya rūpa sukhālambanam asukhavipākam
ihadharme dharmānupaśyī
ādhyātmikeṣu dharmeṣu
cakṣuṣā rūpam ayoniśomanaskāry ālambana paśyati nidhyāpayaty āsvā  dayati 
sukham iti prajānite 
pariṇāmena duḥkhavipāka nara kapretatiryakṣavartanīyam bhavati 
katarat tat karma pratyut pannāsukhavipāki vipariṇāmena sukhavipāki
iha cakṣurvi jñeyāni rūpāni dṛṣṭvā
cakṣuḥsasparśajayoniśomanaskārābhimu khacitto na tatra sarajyate
na mano vidadhāti 
tad asya pratyut pannaduḥkhavipāka bhavati
pariṇāmasukha devamanuṣyo papattaye nirvāṇaparyavasāna ca bhavati 
 eva śrotraghrāṇajihvākāyamanovijñeyeṣu dharmeṣu 
 punar api yogācāra ādhyātmike dharme dharmānupaśyī viharati
katha sa bhikṣuś cakṣurvijñeyeṣu rūpeṣu upekṣako viharati
iha bhikṣuś cakṣuṣā rūpāni dṛṣṭvā
na sarajyate na virajyate na vilokayati na nindati na spṛhayati na manasikurute 
nāyoniśomanasikāreṇāvṛto bhavati 
upekṣako bhavati 
sa upekṣā sthānīya sthāna upalabhate
na duḥkhasukhasthānīyam 
  punar api sa bhikṣur daśama bhūmyantaram ākrāmati ṣaḍbhūmivikārabhūtam
tadyathānāgamyaprathamadhyānabhūmya ntara catvāri dhyānāni 
saḍbhūmivikāram ākrāmati 
sa paśyati dharmāṇām udayavyayam
dharmāṇām udayavyaya jānāti 
nirodhagāminīpratipadāryāṣṭāṅgena mārgeṇa nirvāṇadvāram avaga ntu samprāptaye ca prayatati 
  tasyaiva prayatamānasya mārabandhanāni bhūyiṣṭhaji hmāni dṛṣṭvā
hṛṣṭā bhaumā yakṣā antarīkṣacarāṇā yakṣāṇām abhinivedayanti 
te'pi caturṇṇā mahārājñā abhinivedayanti 
te 'pi cāturmahārājakāyikānā devānām abhinivedayanti 
cātur mahārājakāyikā api tridaśāṇā
tridaśā api yāmānām
yāmā api tuṣitānām
tuṣitā api nirmāṇaratīnām
nirmāṇaratayaḥ paranirmita vaśavartinā
paranirmitavaśavartino brahmakāyikānām abhini vedayanti yathā
jambhūdvīpāt kulaputro  vistareṇa pūrva vad yāvat  aṣṭama bhūmyantaram ārūḍhaḥ
antargataś ca ṣaḍ bhūmayaḥ
tā cārūḍhaḥ 
  tataś ca śrutvā
paramahṛṣṭā brahmakāyikā devā hitvā dhyānasukhavihāram
tadā bhūta śrutvā
hṛṣṭatarā bhavanti  
bhavanti cātra gāthāḥ 
śubhānām aśubhānā ca karmaṇā phalaniścayaḥ 
bhujyate svakṛta sarva karmabaddhā hi dehinaḥ 
 madhurādau vipāke tu kaṭukāḥ kleśabhūmayaḥ 
varjanīyā hi viṣavat sarvānarthakarā hi tāḥ 
  na kleśānā vaśa gacched anjñānasya co gocaram 
jñānavān ihaloke ca pare ca sukham aśnute 
 jñāna vadhyāḥ sadā kleśā agnivadhya yathā tṛṇam 
tasmāj jñāna para brahma ratnatrayavidarśakam 
 ye jñānagocararatās te śānte vartmani sthitāḥ 
kleśoragais tu ye daṣṭās te sarve vilaya gatāḥ 
 parāparajñā ye dhīrāḥ sādhavas tattvadarśinaḥ 
te yānti parama sthāna jarāmaraṇavarjita 
 sasārābhiratā ye tu ramante kleśaśatrubhiḥ 
nityabandhanabaddhās te bhramanti bhavasaṅkaṭe 
 yasya nairyāṇikī buddhir yasya caryā śivā sadā 
tasya devanikā yasya brahmaloko vidhīyate 
 yasya rāgādayo dveṣyāḥ pūjyā buddhādayaḥ sadā 
sa nāśayati sasāra śuṣkendhanam ivānalaḥ 
  yo na cittasya vaśagaś citta yasyānuga sadā 
sa nirnāśayati kleśās tamaḥ sūryodaye yathā 
 cittaśatruḥ para śatrur na śatrur aparaḥ smṛtaḥ 
cittadagdhāḥ sadā sattvāh kāladagdhā yathā nagāḥ 
 yaś cittavaśam āpanno bālo mūḍho'jitendriyaḥ 
tasya duḥkhaśamo nāsti nirvāṇa tasya dūrataḥ 
 duḥkha duḥkhavipāka ca duḥkhahetu vijānataḥ 
naśyanti bandhanāḥ sarve kleśavṛndāḥ samantataḥ 
 ālokānā para jñāna tamasā moha ucyate 
tasmād ālokanirato yaḥ sa paṇḍita ucyate 
 moha vivarjayed dhīmān sarvānarthakaro hi yaḥ 
yo mohavaśam āpannas tasya śāntir na vidyate 
 saspṛśeta vara vahni savāsed uragair varam 
na kleśaiḥ saha sayujyed yadīcchec chreyam ātmanaḥ 
  amṛtānā para jñāna śreya sā nidhir uttamaḥ 
bandhūnā ca paro bandhur dhanānā dhanam uttamam 
 tasmāj jñānāgninā nitya nirdahet kleśaparvatān 
kleśaparvatadagdhasya sukha padam avasthitam 
 andhās tamasi majjante puruṣā mandamedhasaḥ 
ye nodvijanti sasārād adharmaraṇapañjarāt 
 teṣā hi saphala janma yeṣā buddhir avañcitā 
te ca pūjyāḥ sadā sadbhir yeṣā dharme sadā matiḥ     
iti sa bhikṣur eva dharmādharmavidhijñaḥ sadbhūtvā viharati 
tasyaiva sunirmalacetasaḥ
anekāni janmaśatasahasrāṇi śikharāṇi sasāraparvatād viśīryante vāntībhavanti naśyanti
apunar bhavatīti 
 naśyanti cāsya kleśaśatravaḥ
antike cāsya bhavati nirvāṇam
