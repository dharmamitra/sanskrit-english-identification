Dhvajāgrasūtra
Sand_1 evaṁ mayā śrutam / ekasmiṁ samaye bhagavāṁ śrāvastyāṁ viharati sma jetavane anāthapiṇḍadasyārāme /
Sand_2 tatra bhagavāṁ bhikṣūn āmantrayate sma /
Sand_3 sati vo bhikṣava araṇyagatānāṁ vā vṛkṣamūlagatānāṁ vā śunyāgāragatānāṁ vā utpadyeta bhayaṁ vā chambitatvaṁ vā romaharṣo vā /
Sand_4 māṁ yūyaṁ tasmiṁ samaye ākārataḥ samanusmarata /
Sand_5 iti hi sa bhagavāṁ tathāgato ’rhāṁ samyaksaṁbuddho vidyācaraṇasampanna sugato lokavid anuttaraḥ puruṣadamyasārathi śāstā devamanuṣyāṇāṁ  #
buddho bhagavāṁ / 
Sand_6 māṁ vas tasmiṁ samaye ākārataḥ samanusmaratām / yad utpatsyate bhayaṁ vā chambitatvaṁ vā romaharṣo vā tat prahāsyate /
Sand_7 bhūtapūrvaṁ bhikṣava śakro devendro devāsurasaṁgrāme pratyupasthite devāms trāyastriṁśān āmantrayati /
Sand_8 sati vo māriṣā devāsurasaṁgrāme samabhirūḍhāṇāṁ utpadyeta bhayaṁ vā chambitatvaṁ vā romaharṣo vā /
Sand_9 mama vas tasmiṁ samaye vaijayanto dhvajāgraḥ samanusmartavyaḥ / mama vas tasmiṁ samaye
vaijayantaṁ dhvajāgraṁ samanusmaratāṁ / yad utpatsyate bhayaṁ vā chambitatvaṁ vā
romaharṣo vā tat prahāsyate /
Sand_10 evam eva sati vo bhikṣava araṇyagatānāṁ vā vṛkṣamūlagatānāṁ vā śunyāgāragatānāṁ vā utpadyeta bhayaṁ vā chambitatvaṁ vā romaharṣo vā 
Sand_11 māṁ yūyaṁ tasmiṁ samaye ākārataḥ samanusmarata / iti hi sa bhagavāṁ yāvad buddho bhagavāṁ /
Sand_12 māṁ vas tasmiṁ samaye ākārataḥ samanusmaratāṁ yad utpatsyate bhayaṁ vā chambitatvaṁ vā romaharṣo vā tat prahāsyate /
Sand_13 tat kasmād dheto / sa hi śakro devendraḥ sarāgaḥ sadveṣaḥ samoha aparimukto jātijarāvyādhimaraṇaśokaparidevaduḥkhadaurmanasyopāyāsebhyaḥ  #
aparimukto duḥkhād iti vadāmi / 
Sand_14 bhīru chambī utrāsī palāyī sa ekatā bibhety api utrāsity api palāyaty api /
Sand_15 ahaṁ khalu tathāgato loke utpanna arhāṁ samyaksaṁbuddho vidyācaraṇasaṁpannaḥ sugato lokavid anuttaraḥ puruṣadamyasārathi śāstā devamanuṣyāṇāṁ  #
buddho bhagavāṁ / 
Sand_16 vigatarāgo vigatadveṣo vigatamohaḥ parimukto jātijarāvyādhimaraṇaśokaparidevaduḥkhadaurmanasyopāyāsebhya parimukto duḥkhād iti vadāmi 
Sand_17 abhīrur acchaṁbī anutrāsī apalāyī /
Sand_18 tasya mama bhikṣava abhīruṇa acchaṁbina anutrāsina apalāyinaḥ śrotavyaṁ vacanaṁ kartavyaṁ anuśāsanaṁ karaṇīyam idaṁ sthānam iti /
Sand_19 idam avocad bhagavāṁ āpatamanasas te bhikṣavo bhagavato bhāṣitam abhyanandam /
Sand_20 dhvajāgrasūtraṁ / samāptaḥ //
