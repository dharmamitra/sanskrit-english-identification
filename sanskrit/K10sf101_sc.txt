Rāṣṭrapālasūtra
bhagavāṁ kuruṣu janapadeṣu caryāṁ caran sthūlakoṣṭhakam anuprāptāḥ / sthūlakoṣṭhake viharati sthūlakoṣṭhakavanaṣaṇḍe /
aśrauṣuḥ sthūlakoṣṭhīyakā brāhmaṇagṛhapatayo bhagavāṁ kuruṣu janapadeṣu caryāṁ caran
sthūlakoṣṭhakam anuprāptāḥ / sthūlakoṣṭhakam anuprāptaḥ / sthūlakoṣṭhake viharati
sthūlakoṣṭhakavanaṣaṇḍe / Mat_0 /
śrutvā ca punaḥ saṁghāt saṁghaṁ pūgāt pūgaṁ saṁghaṁ pūgāt pūgaṁ saṁgamya samāgamya
sthūlakoṣṭhakān niṣkramya yena bhagavāṁs tenopajagmuḥ upetya bhagavatpādau śirasā
vanditvā ekānte nyaṣīdaṁ / Mat_1 /
ekānte niṣaṇṇāṁ sthūlakoṣṭhīyakāṁ brāhmaṇagṛhapatīṁ bhagavāṁ dharmyā kathayā saṁdarśayati samādāpayati samuttejayati saṁpraharṣayati / Mat_2 
tena khalu samayena rāṣṭrapālo gṛhapatiputras tasmiṁ pariṣadi sanniṣaṇṇo ’bhūt saṁnipatitaḥ / Mat_3 /
atha rāṣṭrapālasya gṛhapatiputrasyaitad abhavad / yathā khalv ahaṁ bhagavato bhāṣitasyārtham
ājānāmi naitat sukaram agāramadhyāvasatā ekāntadhavalīkṛtaṁ yāvajjīvaṁ kevalaṁ paripūrṇaṁ
pariśuddhaṁ paryavadātaṁ brahmacaryaṁ caritum / yannv ahaṁ keśaśmaśruṇy avatārya
kāṣāyāṇi vastrāṇy ācchādya saṁmyag eva śraddhayā agārād anagārikaṁ pravrajeyaṁ / Mat_4 /
atha bhagavāṁ sthūlakoṣṭhīyakāṁ brahmaṇagṛhapatīm anekaparyāyeṇa dharmyā kathayā saṁdarśayitvā samākāpayitvā samuttejayitvā saṁpraharṣayitvā  #
tūṣṇīm abhavat / Mat_5 / 
atha sthūlakoṣṭhīyakā brāhmaṇagṛhapatayo bhagavato bhāṣitam abhinandyānumodya bhagavatpādau śirasā vanditvā bhagavato ’ntikāt prakrāntāḥ / rāṣṭrapālo  #
gṛhapatiputras tatraivāsthāt / Mat_6 / 
atha rāṣṭrapālo gṛhapatiputraḥ aciraprakrāntān sthūlakoṣṭhīyakāṁ brāhmaṇagṛhapatīṁ
viditvā / utthāyāsanād ekaṁsam uttarāsaṁgaṁ kṛtvā yena bhagavāṁs tenāṁ jaliṁ praṇamya
bhagavantam idam avocat / labheyāhaṁ bhadanta svākhyāte dharmavinaye pravrajyām upasaṁpadaṁ
bhikṣubhāvaṁ / careyam ahaṁ bhagavato ’ntike brahmacaryaṁ / Mat_7 /
anujñāto ’si gṛhapatiputramātāpitṛbhyāṁ / Mat_8 /
no bhadanta / Mat_9 /
na hi gṛhapatiputra tathāgata vā tathāgataśrāvaka vāna anujñātaṁ mātāpitṛbhyāṁ pravrajayanti upasaṁpādayanti vā / Mat_10 /
ahaṁ bhadanta tathā kariṣyāmi yathā mātāpitaram anujñāsyati / Mat_11 /
atha rāṣṭrapālo gṛhapatiputro bhagavatpādau śirasā vanditvā bhagavato ’ntikāt prakrāntaḥ / Mat_12 /
atha rāṣṭrapālo gṛhapatiputraḥ svakaṁ niveśanaṁ gatvā mātāpitaraṁ āmantrayati / Mat_13 /
aṁba tāta anujānīdhvaṁ māṁ pravrajiṣyāmi saṁmyag eva śraddhayā agārād anagārikaṁ / Mat_14 /
yat khalu tāta rāṣṭrapāla jānīyāḥ tvam asmākam ekaputrakaḥ priyo manāpaḥ kṣānta apratikūla maraṇenāpi tvayā vayam akāmakā vinā bhaviṣyāmaḥ kutaḥ  #
punar jīvaṁtam utsrakṣyāmaḥ / Mat_15 / 
saced anujñāsyatha ity evaṁ kuśalaṁ nānujñāsyatha adyāgreṇa na annam api bhokṣyāmi / Mat_16 /
atha rāṣṭrapālo gṛhapatiputraḥ ekam api bhaktacchedam akārṣīt / dvau trīn yāvat sapta bhaktacchedānām akārṣīt / Mat_17 /
atha rāṣṭrapālasya gṛhapatiputrasya mātāpitaraṁ rāṣṭrapālaṁ gṛhapatiputram idam avocat / Mat_18 /
yat khalu tāta rāṣṭrapāla jānīyāḥ tvaṁ hi sukumāraḥ sukhaiṣīna tvaṁ jānīṣe duḥkhasya
duṣkaraṁ brahmacaryaṁ duṣkaraṁ prāvivekyaṁ duṣkaraṁ ekatvam abhiramaṁ durabhisaṁbodhāny
araṇyavanaprasthāni prāntāni śayanāsa nāny abhyāvastum / ihaiva tvaṁ tāta rāṣṭrapāla
niṣadya kāmāṁś ca paribhuṁkṣva dānāni ca dehi puṇyāni ca kuru / Mat_19 /
evam ukto rāṣṭrapālo gṛhapatiputras tūṣṇīm / Mat_20 /
atha rāṣṭrapālasya gṛhapatiputrāsya mātāpitaraṁ jñātīn udyojayati / aṁga tāvajjñātayaḥ tātaṁ rāṣṭrapālam utthāpayata / Mat_21 /
atha jñātayo yena rāṣṭrapālo gṛhapatiputras tenopasaṁkrāntāḥ / upetya rāṣṭrapālaṁ gṛhapatiputram idam avocaṁ / Mat_22 /
yat khalu tāta rāṣṭrapāla jānīyās tvaṁ hi sukumāraḥ sukhaiṣī na tvaṁ jānīṣe duḥkhasya pūrvavat puṇyāni ca kuru / Mat_23 /
evam ukto rāṣṭrapālo gṛhapatiputras tūṣṇīṁ / Mat_24 /
atha rāṣṭrapālasya gṛhapatiputrasya mātāpitaraṁ vayasyakān udyojayati / aṁga tāvad vayasyakāḥ tātaṁ rāṣṭrapālam utthāpayata / Mat_25 /
atha rāṣṭrapālasya gṛhapatiputrasya vayasyakāḥ yena raṣṭrapālo gṛhapatiputras tenopasaṁkrāntāḥ / upetya rāṣṭrapālaṁ gṛhapatiputram idam avocaṁ  
yat khalu saumya rāṣṭrapāla jānīyās tvaṁ hi sukumāraḥ sukhaiṣī na tvaṁ jānīṣe duḥkhasya pūrvavat puṇyāni ca kuru / Mat_27 /
evam ukto rāṣṭrapālo gṛhapatiputras tūṣṇīṁ / Mat_28 /
atha rāṣṭrapālasya gṛhapatiputrasya vayasyakā yena rāṣṭrapālasya gṛhapatiputrasya mātāpitaraṁ tenopasaṁkrāntāḥ / upetya rāṣṭrapālasya gṛhapatiputrasya  #
mātāpitaram idam avocaṁ / Mat_29 / 
amba tāta anujānīdhvaṁ putraṁ rāṣṭrapālaṁ pravrajituṁ kiṁ mṛtena kariṣyatha saced
abhiramiṣyati jīvantam enaṁ drakṣyadhve sacen nābhiiramiṣyati kānyā putrasya gatir
anyatra mātāpitaram eva / Mat_30 /
evaṁ vayaṁ kumārakās tātaṁ rāṣṭrapālam anujānāmaḥ sacet pravrajitvā uddarśayitavyam / Mat_31 /
atha rāṣṭrapālasya gṛhapatiputrasya vayasyakā yena rāṣṭrapālo gṛhapatiputras tenopasaṁkrāntāḥ upetya rāṣṭrapālaṁ gṛhapatiputram idam avocam  
yat khalu saumya rāṣṭrapāla jānīyāḥ / anujñāto ’si mātāpitṛbhyāṁ sacet pravrajya uddarśayitavyaṁ / Mat_33 /
atha rāṣṭrapāla gṛhapatiputro ’nupūrveṇa kāyasya sthāmaṁ ca balaṁ ca saṁjānayya yena
bhagavāṁs tenopajagāma / upetya bhagavatpādau śirasā vanditvā ekānte asthād / ekānte
sthito rāṣṭrapālo gṛhapatiputro bhagavantam idam avocat / Mat_34 /
anujñāto ’smi bhadanta mātāpitṛlabhyām labheyāhaṁ svākhyāte dharmavinaye pravrajyām upasaṁpadaṁ bhikṣubhāvaṁ careyāhaṁ bhagavato ’ntike brahmacaryam  
labdhavān rāṣṭrapālo gṛhapatiputraḥ svākhyāte dharmavinaye pravrajyām upasaṁpadaṁ bhikṣubhāvam / Mat_36 /
atha bhagavān gṛhapatiputraṁ rāṣṭrapālaṁ pravrājayitvā upasaṁpādayitvā yathābhiramyaṁ
sthūlakoṣṭhake vihṛtya yena śrāvastī tena caryāṁ prakrāntaḥ / anupūrveṇa caryāṁ caraṁ
śravastīm anuprāptaḥ / śravastyāṁ viharati jetavana anāthapiṇḍadasya ārāme / Mat_37 /
tena khalu samayenāyuṣmato rāṣṭrapālasya pitāmadhyame dvārasāle muṇḍinam ullekhayati / adrākṣīd āyuṣmato rāṣṭrapālasya pitāyuṣmantaṁ rāṣṭrapālaṁ  #
dvārasamīpe / Mat37J / 
dṛṣṭvaitad avocat / tair muṇḍakaiḥ śramaṇakair asmākaṁ kulasya pratiprasrabdhaṁ kṛtaṁ / Mat37K /
yair asmākam ekaputrakaḥ priyo manāpaḥ kṣānto ’pratikūlo rāṣṭrapālaḥ pravrājayita
upasaṁpādayitas teṣāṁ muṇḍakānāṁ śramaṇakānāṁ kena piṇḍaṁ dātavyaṁ kartavyaṁ manyate / Mat37K /
athāyuṣmān rāṣṭrapālaḥ svake niveśane ākrośād anyaṁ naiva dānam alabdha / pravāsayitas tvaritatvaritaṁ pratiniṣkrāntaḥ / Mat37L /
tena khalu samayenāyuṣmato rāṣṭrapālasya purāṇajīrṇadāsī paryuṣitaṁ pūtikulmāṣam
ākīrya bahir nirgatā cchorayitum / āyuṣmān rāṣṭrapālas tāṁ purāṇajīrṇadāsīm adrākṣīt / Mat37M /
dṛṣṭvā tāṁ jīrṇadāsīm etad avocat / Mat37M /
bhagini sacet paryuṣitāḥ pūtikulmāṣāḥ parityaktāḥ asmākaṁ pātre ākira / āryaṁ yāca / Mat37N /
atha sā purāṇajīrṇadāsī paryuṣitaṁ pūtikulmāṣaṁ pātra ākirantī āyuṣmato rāṣṭrapālasya
hastataś ca pādataś ca ghrāṇataś ca / mukhataś ca svaraguptitaś ca nimittaṁ pratyabhijñāyedam avocat / Mat37O /
āryaptra kiṁ rāṣṭrapālaḥ / bhagini māṁ tathā janaḥ saṁjānāti / Mat37P /
atha sā purāṇajīrṇadāsī śīghraṁsīghraṁ yenāyuṣmato rāṣṭrapālasya mahallakas tenopasaṁkrāntā / upetyāyuṣmato rāṣṭrapālasya mahallakam idam avocat  
svāmi jānīyāḥ saumyo rāṣṭrapālaḥ sthūlakoṣṭakaṁ ciraprāptaḥ svaniveśanaṁ tu prāptuṁ necchati / Mat37R /
athāyuṣmato rāṣṭrapālasya mahallako vāmahastena keśaṁ saṁnipātayan paṭakam ekanivāsaṁ nivāsayya yenāyuṣmān rāṣṭrapālo tenopasaṁkrāntaḥ / Mat37S 
tena khalu samayenāyuṣmān rāṣṭrapālaḥ prakārakaṇṭake sthitas taṁ paryuṣitaṁ pūtikulmāṣaṁ paribhuñjati / Mat37T /
śakṭeṣv āropayya nadyā gaṅgāyāḥ sroto gaṁbhīrataraṁ ca śīghrataraṁ ca tatra prakṣipa mā te atonidānam utpatsyante śokaparidevaduḥkhadaurmanasyopāyāsāḥ  
athāyuṣmato rāṣṭrapālasya mahallikāyāḥ etad abhavad / evam apy ahaṁ kurvatī na śakyā tātaṁ rāṣṭrapālam anvāvartitum / yanv ahaṁ vadhukā udyojayeyam  
athāyuṣmato rāṣṭrapālasya mahallikā vadhukā udyojayantīdam avocat / Mat_40 /
vadhukā / yadrūpeṇa pūrvaṁ snānena snātāḥ vastraiḥ prāvṛtāḥ alaṁkāreṇālaṁkṛtāḥ dhūpair
dhūpitāḥ cūrṇair avacūrṇitā rāṣṭrapālasya pūrve āgārikasya sataḥ priyāś cābhūta manāpāś
ca tadrūpaiḥ snātraiḥ snātāḥ vastraiḥ prāvṛtāḥ alaṁkāreṇālaṁkṛtāḥ dhūpair dhūpitāḥ
cūrṇair avacūrṇitāḥ ekā ekasmiṁ pāde nipatitvā dvitīyā dvitīye nipatitvā evaṁ vadata / Mat_41 /
kathaṁ rūpās tā apsarasa yāsām arthe āryaputro bhagavato ’ntike brahmacaryaṁ carati /
athāyuṣmato rāṣṭrapālasya purāṇadvitīyā yadrūpaiḥ snānaiḥ snātāḥ pūrvavad yāvac cūrṇair
avacūrṇitā rāṣṭrapālasya / pūrve āgārikasya sataḥ priyāś cābhūvan manāpāś ca tadrūpaiḥ
snānaiḥ snātāḥ pūrvavad yāvac cūṇair avacūrṇitāḥ ekā ekasmiṁ pāde nipatitā dvitīyā
dvitīye evam kathaṁ rūpās tā apsaraso yāsām arthe āryaputro bhagavato ’ntike brahmacaryaṁ carati / Mat_42 /
nāhaṁ bhaginyaḥ apsarasāṁ hetor bhagavato ’ntike brahmacaryaṁ carāmi / Mat_43 /
athāyuṣmato rāṣṭrapālasya purāṇadvitīyā bhaginīvādena samudācīrṇe / mūrcchitā bhūmyāṁ nipatitāḥ / Mat_44 /
athāyuṣmāṁ rāṣṭrapālo mātāpitaram āmantrayati / aṁbatāta saced annaṁ parityaktaṁ kiṁ na kīyate kiṁ viheṭhīyate parityakte / Mat_45 /
bhuṁkṣva tāta rāṣṭrapāla / Mat_46 /
athāyuṣmato rāṣṭrapālasya mahallako mahallikā cāyuṣmantaṁ rāṣṭrapālaṁ śucinā praṇītena
khādanīyabhojanīyena svahastaṁ santarpayanti saṁpravārayanti / anekaparyāyeṇa śucinā
praṇītena khādanīyabhojanīyena svahastaṁ santarpayitvā saṁpravārayitvā / Mat_47 /
āyuṣmantaṁ rāṣṭrapālaṁ bhuktavantaṁ viditvā dhautahastam apanītapātraṁ nīcatarakam āsanaṁ gṛhītvā niṣaṇṇā dharmaśravaṇāya / Mat_48 /
athāyuṣmato rāṣṭrapālas tasyāṁ velāyāṁ gāthā babhāṣe / Mat_49 /
grahāyakapaṁñña śrāmaṇyam eva śreyaḥ // Mat_49 /
atho mahārājā kauravya āyuṣmato rāṣṭrapālasya bhāṣitam abhinandyānumodyāyuṣmatpādauśirasā vanditvāsanād utthāya prakrāntaḥ / Mat_49 /
