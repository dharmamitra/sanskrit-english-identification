Mṛtyuvañcanatārāsādhana
// namaḥ sitatārāyai //
pūrvavac chūnyatāparyantaṃ vibhāvya
sitāravindamadhyasthāṃ tāṃbhūtāṃ candraviṣṭarām /
ābaddhavajraparyaṅkāṃ varadotpaladhāriṇīm // Mvts_1 //
śaraccandrakarākārāṃ pṛṣṭhacandrasamāśritām /
sarvālaṅkārasaṃpūrṇāṃ ṣoḍaśābdavapuḥkramām // Mvts_2 //
dhyātvāryatārāṃ hṛdaye tasyāś cakraṃ sitadyuti /
aṣṭakoṣṭhakam aṣṭābhir akṣaraiḥ paripūritam // Mvts_3 //
oṃhāvyañjanamadhyasthasādhyanāmādyanābhikam /
dhyāyād ekāgracittaḥ san ṣaṇ māsān dṛḍhaniścayaḥ // Mvts_4 //
japed akhinnacittaḥ san mantram enaṃ daśākṣaram //
tatrāiṣa mantraḥ oṃ tāre tuttāre ture svāhā // Mvts_5 //
mṛtyudoṣair na liptaḥ syāt tārāhṛccakrabhāvakaḥ /
lābhī ca sarvasiddhīnāṃ ṣaṇmāsāvadhi tatparaḥ // Mvts_6 //
// mṛtyuvañcanatārāsādhanam //
