Māratajjanīyasūtra
WalA.1 śramaṇa anenāvavādenāvavaditā maitrāsahagatena cittena pūrvavad yāvad vyāhārṣuḥ yatraiṣāṁ dūṣī māraḥ avatāraprekṣī avatāragaveṣī na labhate  #
avatāraṁ na labhate ālambanam / 
WalA.2 tena khalu samayena yaḥ kaścic chrāddho brāhamaṇagṛhapatiḥ kālaṁ karoti sa yadbhūyasā kāyasya bhedāt paraṁ maraṇād apāyadurgativivnipātaṁ  #
narakeṣūpapadyate / 
WalA.3 atha dūṣino mārasyāvatāraprekṣino ’vatāragaveṣina etad abhavat / tathāhaṁ na labhe
muṇḍakānāṁ śramaṇānāṁ avatāraṁ na labhe ālambanaṁ yannv ahaṁ śrāddhāṁ brāhmaṇagṛhapatīms
tathā tathā anvāviśaṁ ye yathā śramaṇāṁ śīlavato guṇavato dṛṣṭvā satkurvaṁti gurukurvaṁti
mānayaṁti pūjayaṁti / atra śramaṇānāṁ satkriyatām gurukriyatām pūjyatāṁ syāc cetasa
anyathātvaṁ yathāhaṁ avatāraprekṣī avatāraragaveṣī labhe avatāraṁ labhe ālambanam /
WalA.4 tena khalu samayena śraddhā brāhaṇagṛhapatayaḥ tathā tathā anvāviṣṭā yathā śramaṇāṁ śīlavato guṇavato dṛṣṭvā satkurvaṁti gurukurvaṁti  #
mānayaṁti pūjayaṁti / 
WalA.5 apīdānīṁ śikṣodditā piṇḍapātā brāhmaṇagṛhapatibhiḥ prajñaptāḥ / evaṁ cāhuḥ / pratigṛhṇantv
asmākaṁ śramaṇāḥ śīlavaṁto guṇavaṁtaḥ śikṣodditāṁ piṇḍapātāṁs tad asmākam bhaviṣyati
dīrgharātram arthāya hitāya sukhāya /
WalA.6 apīdānīṁ śramaṇānām ahatāni vastrāṇi navāni dīrghadaśāni rathyāmukheṣu prajñapya
evam āhuḥ / ākramantv asmākaṁ śramaṇāḥ śīlavaṁto guṇavaṁtaḥ ahatāni vastrāṇi navāni
dīrghadaśāni pādbhyāṁ tad asmākaṁ bhaviṣyati dīrgharātram arthāya hitāya sukhāya /
WalA.7 apīdānīṁ śramaṇānām uttamāṅgaruhāṁ keśāṁ rathyāmukheṣu prajñapyaivam āhuḥ / ākramaṁtu
guṇavaṁtaḥ uttamāṅgaruhān keśāṁ pādbhyāṁ tad asmākaṁ bhaviṣyati dīrgharātram arthāya hitāya sukhāya /
WalA.8 apīdānīṁ śramaṇāṁ śīlavato gunavato dṛṣṭvā sarvāṅgair anupariṣvajya antargṛhe praviśya
prajñapta evāsane niṣādayaṁti / evam cāhuḥ / nisīdantv asmākaṁ śramaṇāḥ śīlavaṁto
guṇavaṁtaḥ prajñapta evāsane tad asmākaṁ bhaviṣyati dīrgharātram arthāya hitāya sukhāya /
WalA.9 atha krakasundasya samyaksaṁbuddhasya śrāvakāḥ satkṛtā gurukṛtā mānitā pūjitā yena krakasundaḥ samyaksaṁbuddhas tenopajagmuḥ /
WalB.1 śramaṇa krakasundena samyaksaṁbuddhena anenāvavādenāvavaditāḥ saṁskāreṣv anityānudarśino
viharaṁti pūrvavad yāvat pratiniḥsargānudarśinaḥ / yatraiṣāṁ dūṣī māraḥ avatāraprekṣī
avatāragaveṣī na labhate avatāraṁ na labhate ālambhanam /
WalB.2 tena khalu samayena yaḥ kaścic chrāddho brāhamaṇagṛhapatiḥ kālaṁ karoti sa yadbhūyasā kāyasya bhedāt paraṁ maraṇāt sugatau svargaloke  #
deveṣūpadyate / 
WalB.3 atha dūṣino mārasyāvatāraprekṣina etad abhavat / tathāpy ahaṁ tr … na labhe muṇḍakānāṁ
śramaṇānāṁ evam anudarśinām avatāraṁ na labhe ālambanaṁ / yaṁnv ahaṁ kāyena āyuṣmantaṁ vidūram ākrameya /
WalB.4 eko ’yaṁ samayaḥ krakasundastathāgato ’rhan samyaksaṁbuddhaḥ sālāyāṁ viharati brāhmaṇagrāmake āyuṣmatā vidūreṇa paścācchramaṇena /
WalB.5 atha krakasundaḥ saṁyaksaṁbuddhas tasyā eva rātryā atyayāt pūrvāhṇe nivasya pātracīvaram ādāya brāhmaṇagrāmakaṁ piṇḍāya prāviśad āyuṣmatā  #
vidūreṇa paścācchramaṇena / 
WalB.6 atha dūṣī māraḥ kumārarūpam anvāviśya śarkaram ādāya āyuṣmato vidūrasya śiro bhagnaṁ rudhiraṁ mukhe gālitam /
WalB.7 tathāpy āyuṣmāṁ vidūraḥ śirasā bhagnena rudhireṇa mukhe galatā krakasundaṁ samyaksaṁbuddhaṁ pṛṣṭhataḥ pṛṣṭhataḥ samanubaddhaḥ /
WalB.8 adrākṣīt pāpimaṁ krakasundaḥ samyaksaṁbuddho dakṣiṇena sarvakāyena nāgāvalokitena
avalokayaṁn āyuṣmato vidūrasya śiro bhagnaṁ rudhiraṁ mukhe galat / dṛṣṭvā ca punar
huntiṁ kṛtvā vivācayati / ayaṁ dūṣī māra ṛddhimān / ayam eva dūṣī māro mātrāṁ nājñāsīt /
WalB.9 atra khalu pāpimaṁ yasya kasyacid ananvayāt tathāgatena hunti kṛtvā vivācitaṁ /
WalB.10 atha sa dūṣī māras tasmāt sthānāt samantatareṇa avīcau mahānarake papitaḥ /
WalB.11 tena khalu samayenāvīcer mahānarakasya catasraḥ saṁjñā āsaṁ avīcir avirala iti śataśaṅkur ity api pratyātmavedanīya ity api ṣaṭsparśāyatanīya  #
ity api /
