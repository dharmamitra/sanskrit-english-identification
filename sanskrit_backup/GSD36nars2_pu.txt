mātṛkā 1
[vyavahāraḥ]
dharmaikatānāḥ puruṣā yadāsan satyavādinaḥ /
tadā na vyavahāro 'bhūn na dveṣo nāpi matsaraḥ // Nar_M1.1 //
naṣṭe dharme manuṣyeṣu vyavahāraḥ pravartate /
draṣṭā ca vyavahārāṇāṃ rājā daṇḍadharaḥ kṛtaḥ // Nar_M1.2 //
likhitaṃ sākṣiṇaś cātra dvau vidhī saṃprakīrtitau /
saṃdigdhārthaviśuddhyarthaṃ dvayor vivadamānayoḥ // Nar_M1.3 //
sottaro 'nuttaraś caiva sa vijñeyo dvilakṣaṇaḥ /
sottaro 'bhyadhiko yatra vilekhāpūrvakaḥ paṇaḥ // Nar_M1.4 //
vivāde sottarapaṇe dvayor yas tatra hīyate /
sa paṇaṃ svakṛtaṃ dāpyo vinayaṃ ca parājaye // Nar_M1.5 //
sāras tu vyavahārāṇāṃ pratijñā samudāhṛtā /
taddhānau hīyate vādī taraṃs tām uttaro bhavet // Nar_M1.6 //
kulāni śreṇayaś caiva gaṇāś cādhikṛto nṛpaḥ /
pratiṣṭhā vyavahārāṇāṃ gurvebhyas tūttarottaram // Nar_M1.7 //
sa catuṣpāc catuḥsthānaś catuḥsādhana eva ca /
caturhitaś caturvyāpī catuṣkārī ca kīrtyate // Nar_M1.8 //
aṣṭāṅgo 'ṣṭādaśapadaḥ śataśākhas tathaiva ca /
triyonir dvyabhiyogaś ca dvidvāro dvigatis tathā // Nar_M1.9 //
dharmaś ca vyavahāraś ca caritraṃ rājaśāsanam /
catuṣpād vyavahāro 'yam uttaraḥ pūrvabādhakaḥ // Nar_M1.10 //
tatra satye sthito dharmo vyavahāras tu sākṣiṣu /
caritraṃ pustakaraṇe rājājñāyāṃ tu śāsanam // Nar_M1.11 //
sāmādyupāyasādhyatvāc catuḥsādhana ucyate /
caturṇām āśramāṇāṃ ca rakṣaṇāt sa caturhitaḥ // Nar_M1.12 //
kartṝn atho sākṣiṇaś ca sabhyān rājānam eva ca /
vyāpnoti pādaśo yasmāc caturvyāpī tataḥ smṛtaḥ // Nar_M1.13 //
dharmasyārthasya yaśaso lokapaktes tathaiva ca /
caturṇāṃ karaṇād eṣāṃ catuṣkārī prakīrtitaḥ // Nar_M1.14 //
rājā sapuruṣaḥ sabhyāḥ śāstraṃ gaṇakalekhakau /
hiraṇyam agnir udakam aṣṭāṅgaḥ sa udāhṛtaḥ // Nar_M1.15 //
ṛṇādānaṃ hy upanidhiḥ saṃbhūyotthānam eva ca /
dattasya punar ādānam aśuśrūṣābhyupetya ca // Nar_M1.16 //
vetanasyānapākarma tathaivāsvāmivikrayaḥ /
vikrīyāsaṃpradānaṃ ca krītvānuśaya eva ca // Nar_M1.17 //
samayasyānapākarma vivādaḥ kṣetrajas tathā /
strīpuṃsayoś ca saṃbandho dāyabhāgo 'tha sāhasam // Nar_M1.18 //
vākpāruṣyaṃ tathaivoktaṃ daṇḍapāruṣyam eva ca /
dyūtaṃ prakīrṇakaṃ caivety aṣṭādaśapadaḥ smṛtaḥ // Nar_M1.19 //
eṣām eva prabhedo 'nyaḥ śatam aṣṭottaraṃ smṛtam /
kriyābhedān manuṣyāṇāṃ śataśākho nigadyate // Nar_M1.20 //
kāmāt krodhāc ca lobhāc ca tribhyo yasmāt pravartate /
triyoniḥ kīrtyate tena trayam etad vivādakṛt // Nar_M1.21 //
dvyabhiyogas tu vijñeyaḥ śaṅkātattvābhiyogataḥ /
śaṅkāsatāṃ tu saṃsargāt tattvaṃ hoḍhādidarśanāt // Nar_M1.22 //
pakṣadvayābhisaṃbandhād dvidvāraḥ samudāhṛtaḥ /
pūrvavādas tayoḥ pakṣaḥ pratipakṣas taduttaram // Nar_M1.23 //
bhūtacchalānusāritvād dvigatiḥ sa udāhṛtaḥ /
bhūtaṃ tattvārthasaṃyuktaṃ pramādābhihitaṃ chalam // Nar_M1.24 //
tatra śiṣṭaṃ chalaṃ rājā marṣayed dharmasādhanaḥ /
bhūtam eva prapadyeta dharmamūlā yataḥ śriyaḥ // Nar_M1.25 //
dharmeṇoddharato rājño vyavahārān kṛtātmanaḥ /
saṃbhavanti guṇāḥ sapta sapta vahner ivārciṣaḥ // Nar_M1.26 //
dharmaś cārthaś ca kīrtiś ca lokapaktir upagrahaḥ /
prajābhyo bahumānaś ca svarge sthānaṃ ca śāśvatam // Nar_M1.27 //
tasmād dharmāsanaṃ prāpya rājā vigatamatsaraḥ /
samaḥ syāt sarvabhūteṣu bibhrad vaivasvataṃ vratam // Nar_M1.28 //
dharmaśāstraṃ puraskṛtya prāḍvivākamate sthitaḥ /
samāhitamatiḥ paśyed vyavahārān anukramāt // Nar_M1.29 //
āgamaḥ prathamaṃ kāryo vyavahārapadaṃ tataḥ /
vivitsā nirṇayaś caiva darśanaṃ syāc caturvidham // Nar_M1.30 //
dharmaśāstrārthaśāstrābhyām avirodhena mārgataḥ /
samīkṣamāṇo nipuṇaṃ vyavahāragatiṃ nayet // Nar_M1.31 //
yathā mṛgasya viddhasya vyādho mṛgapadaṃ nayet /
kakṣe śoṇitaleśena tathā dharmapadaṃ nayet // Nar_M1.32 //
yatra vipratipattiḥ syād dharmaśāstrārthaśāstrayoḥ /
arthaśāstroktam utsṛjya dharmaśāstroktam ācāret // Nar_M1.33 //
dharmaśāstravirodhe tu yuktiyukto 'pi dharmataḥ /
vyavahāro hi balavān dharmas tenāvahīyate // Nar_M1.34 //
sūkṣmo hi bhagavān dharmaḥ parokṣo durvicāraṇaḥ /
ataḥ pratyakṣamārgeṇa vyavahāragatiṃ nayet // Nar_M1.35 //
yāty acauro 'pi cauratvaṃ cauraś cāyāty acauratām /
acauraś cauratāṃ prāpto māṇḍavyo vyavahārataḥ // Nar_M1.36 //
strīṣu rātrau bahir grāmād antarveśmany arātiṣu /
vyavahāraḥ kṛto 'py eṣu punaḥ kartavyatām iyāt // Nar_M1.37 //
gahanatvād vivādānām asāmarthyāt smṛter api /
ṛṇādiṣu haret kālaṃ kāmaṃ tattvabubhutsayā // Nar_M1.38 //
gobhūhiraṇyastrīsteya- pāruṣyātyayikeṣu ca /
sāhaseṣv abhiśāpe ca sadya eva vivādayet // Nar_M1.39 //
anāvedya tu yo rājñe saṃdigdhe 'rthe pravartate /
prasahya sa vineyaḥ syāt sa cāsyārtho na sidhyati // Nar_M1.40 //
vaktavye 'rthe na tiṣṭhantam utkrāmantaṃ ca tadvacaḥ /
āsedhayed vivādārthī yāvad āhvānadarśanam // Nar_M1.41 //
sthānāsedhaḥ kālakṛtaḥ pravāsāt karmaṇas tathā /
caturvidhaḥ syād āsedho nāsiddhas taṃ vilaṅghayet // Nar_M1.42 //
nadīsaṃtārakāntāra- durdeśopaplavādiṣu /
āsiddhas taṃ parāsedham utkrāman nāparādhnuyāt // Nar_M1.43 //
āsedhakāla āsiddha āsedham yo vyatikramet /
sa vineyo 'nyathā kurvann āseddhā daṇḍabhāg bhavet // Nar_M1.44 //
nirveṣṭukāmo rogārto yiyakṣur vyasane sthitaḥ /
abhiyuktas tathānyena rājakāryodyatas tathā // Nar_M1.45 //
gavāṃ pracāre gopālāḥ sasyabandhe kṛṣīvalāḥ /
śilpinaḥ cāpi tatkālam āyudhīyāś ca vigrahe // Nar_M1.46 //
aprāptavyavahāraś ca dūto dānonmukho vratī /
viṣamasthaś ca nāsedhyo na cainān āhvayen nṛpaḥ // Nar_M1.47 //
nābhiyukto 'bhiyuñjīta tam atīrtvārtham anyataḥ /
na cābhiyuktam anyena na viddhaṃ veddhum arhati // Nar_M1.48 //
yam artham abhiyuñjīta na taṃ viprakṛtiṃ nayet /
nānyat pakṣāntaraṃ gacched gacchan pūrvāt sa hīyate // Nar_M1.49 //
na ca mithyābhiyuñjīta doṣo mithyābhiyoginaḥ /
yas tatra vinayaḥ proktaḥ so 'bhiyoktāram āvrajet // Nar_M1.50 //
sāpadeśaṃ haran kālam abruvaṃś cāpi saṃsadi /
uktvā vaco vibruvaṃś ca hīyamānasya lakṣaṇam // Nar_M1.51 //
palāyate ya āhūtaḥ prāptaś ca vivaden na yaḥ /
vineyaḥ sa bhaved rājñā hīna eva sa vādataḥ // Nar_M1.52 //
nirṇiktavyavahāreṣu pramāṇam aphalaṃ bhavet /
likhitaṃ sākṣiṇo vāpi pūrvam āveditaṃ na cet // Nar_M1.53 //
yathā pakveṣu dhānyeṣu niṣphalāḥ prāvṛṣo guṇāḥ /
nirṇiktavyavahārāṇāṃ pramāṇam aphalaṃ tathā // Nar_M1.54 //
abhūtam apy abhihitaṃ prāptakālaṃ parīkṣyate /
yat tu pramādān nocyeta tad bhūtam api hīyate // Nar_M1.55 //
tīritaṃ cānuśiṣṭaṃ ca yo manyeta vidharmataḥ /
dviguṇaṃ daṇḍam āsthāya tat kāryaṃ punar uddharet // Nar_M1.56 //
durdṛṣṭe vyavahāre tu sabhyās taṃ daṇḍam āpnuyuḥ /
na hi jātu vinā daṇḍaṃ kaścin mārge 'vatiṣṭhate // Nar_M1.57 //
rāgād ajñānato vāpi lobhād vā yo 'nyathā vadet /
sabhyo 'sabhyaḥ sa vijñeyas taṃ rājā vinayed bhṛśam // Nar_M1.58 //
kiṃtu rājñā viśeṣeṇa svadharmam anurakṣatā /
manuṣyacittavaicitryāt parīkṣyā sādhvasādhutā // Nar_M1.59 //
puruṣāḥ santi ye lobhāt prabrūyuḥ sākṣyam anyathā /
santi cānye durātmānaḥ kūṭalekhyakṛto janāḥ // Nar_M1.60 //
ataḥ parīkṣyam ubhayam etad rājñā viśeṣataḥ /
lekhyācāreṇa likhitaṃ sākṣyācāreṇa sākṣiṇaḥ // Nar_M1.61 //
asatyāḥ satyasaṃkāśāḥ satyāś cāsatyadarśanāḥ /
dṛśyante vividhā bhāvās tasmād yuktaṃ parīkṣaṇam // Nar_M1.62 //
talavad dṛśyate vyoma khadyoto havyavāḍ iva /
na talaṃ vidyate vyomni na khadyote hutāśanaḥ // Nar_M1.63 //
tasmāt pratyakṣadṛṣṭo 'pi yuktam arthaḥ parīkṣitum /
parīkṣya jñāpayan arthān na dharmāt parihīyate // Nar_M1.64 //
evaṃ paśyan sadā rājā vyavahārān samāhitaḥ /
vitatyeha yaśo dīptaṃ bradhnasyāpnoti viṣṭapam // Nar_M1.65 //
mātṛkā 2 ( mātṛkā 2 is found only in ms P. )
[The second chapter of the mātṛkā included in Jolly's edition and translation
is not to be part of the original Nāradasmṛti (see the Introduction to the text).
Its translation is given in the Appendix.]
[bhāṣā]
suniścitabalādhānas tv arthī svārthapracoditaḥ /
lekhayet pūrvapakṣaṃ tu kṛtakāryaviniścayaḥ // Nar_M2.1 //
pūrvapakṣaśrutārthas tu pratyarthī tadanantaram /
pūrvapakṣārthasaṃbandhaṃ pratipakṣaṃ niveśayet // Nar_M2.2 //
śvo lekhanaṃ vā sa labhet tryahaṃ saptāham eva vā /
arthī tṛtīyapāde tu yuktaṃ sadyo dhruvaṃ jayī // Nar_M2.3 //
mithyā saṃpratipattir vā pratyavaskandam eva vā /
prāṅnyāyavidhisādhyaṃ vā uttaraṃ syāc caturvidham // Nar_M2.4 //
mithyaitan nābhijānāmi tadā tatra na saṃnidhiḥ /
ajātaś cāsmi tatkāla evaṃ mithyā caturvidhā // Nar_M2.5 //
mithyā ca viparītaṃ ca punaḥ śabdasamāgamam /
pūrvapakṣārthasaṃbandham uttaraṃ syāc caturvidham // Nar_M2.6 //
bhāṣāyā uttaraṃ yāvat pratyarthī na niveśayet /
arthī tu lekhayet tāvad yāvad vastu vivakṣitam // Nar_M2.7 //
anyārtham arthahīnaṃ ca pramāṇāgamavarjitam /
lekhyaṃ hīnādhikaṃ bhraṣṭaṃ bhāṣādoṣās tūdāhṛtāḥ // Nar_M2.8 //
labdhavyaṃ yena yad yasmāt sa tat tasmād avāpnuyāt /
na tv anyo 'nyad athānyasmād ity anyārtham idaṃ tridhā // Nar_M2.9 //
manasāham api dhyātas tvanmitreṇeha śatruvat /
ato 'nyathā mahākṣāntyā tvam ihāvedito mayā // Nar_M2.10 //
dravyapramāṇahīnaṃ yat phalopāśrayavarjitam /
pramāṇavarjitaṃ nāma lekhyadoṣaṃ tad utsṛjet // Nar_M2.11 //
āgamavarjitaṃ doṣaṃ pūrvapāde vivarjayet /
ekasya bahubhiḥ sārdhaṃ purarāṣṭravirodhakam // Nar_M2.12 //
bindumātrāpadavarṇeṣv ekāvidhiṣṭayā (?) /
hīnādhikā bhaved vyarthā tāṃ yatnena vivarjayet // Nar_M2.13 //
bhraṣṭaṃ tu duḥsthitaṃ yat syāj jalatailādibhir hatam /
bhāṣāyāṃ tad api spaṣṭaṃ vispaṣṭārthaṃ vivarjayet // Nar_M2.14 //
satyā bhāṣā na bhavati yady api syāt pratiṣṭhitā /
bahiś ced bhraśyate dharmān niyatād vyavahārikāt // Nar_M2.15 //
gandhamādanasaṃsthasya mayāsyāsīt tad arpitam /
vyavahārikadharmasya bāhyam etan na sidhyati // Nar_M2.16 //
anyākṣaraniveśena anyārthagamanena ca /
ākulaṃ ca kriyādānaṃ kriyā caivākulā bhavet // Nar_M2.17 //
rāgādīnāṃ yad ekena kopitaḥ karaṇe vadet /
tad ādau tu likhet sarvaṃ vādinaḥ phalakādiṣu // Nar_M2.18 //
nirākulāvabodhāya dharmasthaiḥ suvicāritam /
tasmād anyad vyapohyaṃ syād vādinaḥ phalakādiṣu // Nar_M2.19 //
vādibhyām abhyanujñātaṃ śeṣaṃ ca phalake sthitam /
sasākṣikaṃ likheyus te pratipattiṃ ca vādinoḥ // Nar_M2.20 //
vādibhyāṃ likhitāc cheṣaṃ yat punar vādinā smṛtam /
tat pratyākalitaṃ nāma svapāde tasya likhyate // Nar_M2.21 //
arthinā saṃniyukto vā pratyarthiprahito 'pi vā /
yo yasyārthe vivadate tayor jayaparājayau // Nar_M2.22 //
yo na bhrātā na ca pitā na putro na niyogakṛt /
parārthavādī daṇḍyaḥ syād vyavahāre 'pi vibruvan // Nar_M2.23 //
pūrvavādaṃ parityajya yo 'nyam ālambate punaḥ /
vādasaṃkramaṇāj jñeyo hīnavādī sa vai naraḥ // Nar_M2.24 //
sarveṣv api vivādeṣu vākchale nāpahīyate /
paśustrībhūmyṛṇādāne śāsyo 'py arthān na hīyate // Nar_M2.25 //
abhiyukto 'bhiyogasya yadi kuryād apahnavam /
abhiyoktā diśed deśyaṃ pratyavaskandito na cet // Nar_M2.26 //
pūrvapāde hi likhitaṃ yathākṣaram aśeṣataḥ /
arthī tṛtīyapāde tu kriyayā pratipādayet // Nar_M2.27 //
kriyāpi dvividhā proktā mānuṣī daivikī tathā /
mānuṣī lekhyasākṣibhyāṃ dhaṭādir daivikī smṛtā // Nar_M2.28 //
divā kṛte kāryavidhau grāmeṣu nagareṣu vā /
saṃbhave sākṣiṇāṃ caiva divyā na bhavati kriyā // Nar_M2.29 //
araṇye nirjane rātrāv antarveśmani sāhase /
nyāsasyāpahnave caiva divyā saṃbhavati kriyā // Nar_M2.30 //
kāraṇapratipattyā ca pūrvapakṣe virodhite /
abhiyuktena vai bhāvyaṃ vijñeyaṃ pūrvapakṣavat // Nar_M2.31 //
palāyate ya āhūto maunī sākṣiparājitaḥ /
svayam abhyupapannaś ca avasannaś caturvidhaḥ // Nar_M2.32 //
anyavādī kriyādveṣī nopasthātā niruttaraḥ /
āhūtaprapalāyī ca hīnaḥ pañcavidhaḥ smṛtaḥ // Nar_M2.33 //
maṇayaḥ padmarāgādyā dīnārādi hiraṇmayam /
muktāvidrumaśaṅkhādyāḥ praduṣṭāḥ svāmigāminaḥ // Nar_M2.34 //
gandhamālyam adattaṃ tu bhūṣaṇaṃ vāsa eva vā /
pāduketi rājoktaṃ tad ākrāman vadham arhati // Nar_M2.35 //
paṇyamūlyaṃ bhṛtir nyāso daṇḍo yac cāvahārakam /
vṛthādānākṣikapaṇā vardhante nāvivakṣitāḥ // Nar_M2.36 //
mithyābhiyogino ye syur dvijānāṃ śūdrayonayaḥ /
teṣāṃ jihvāṃ samutkṛtya rājā śūle vidhāpayet // Nar_M2.37 //
ājñā lekhaḥ paṭṭakaḥ śāsanaṃ vā ādhiḥ pattraṃ vikrayo vā krayo vā /
rājñe kuryāt pūrvam āvedanaṃ yas tasya jñeyaḥ pūrvapakṣaḥ vidhijñaiḥ // Nar_M2.38 //
sākṣikadūṣaṇe kāryaṃ pūrvasākṣiviśodhanam /
śuddheṣu sākṣiṣu tataḥ paścāt sākṣyaṃ viśodhayet // Nar_M2.39 //
sākṣisabhyāvasannānāṃ dūṣaṇe darśanaṃ punaḥ /
svacaryāvasitānāṃ tu nāsti paunarbhavo vidhiḥ // Nar_M2.40 //
svayam abhyupapanno 'pi svacaryāvasito 'pi san /
kriyāvasanno 'py arheta paraṃ sabhyāvadhāraṇam // Nar_M2.41 //
pakṣān utsārya kāryas tu sabhyaiḥ kāryaviniścayaḥ /
anutsāritanirṇikte virodhaḥ pretya ceha ca // Nar_M2.42 //
sabhair eva jitaḥ paścād rājñā śāsyaḥ svaśāstrataḥ /
jayine cāpi deyaṃ syād yathāvaj jayapatrakam // Nar_M2.43 //
vyavahāramukhaṃ caitat pūrvam uktaṃ svayaṃbhuvā /
mukhaśuddhau hi śuddhiḥ syād vyavahārasya nānyathā // Nar_M2.44 //
mātṛkā 3
[sabhā]
niyuktena tu vaktavyam apakṣapatitaṃ vacaḥ // Nar_M3.1 //
yuktarūpaṃ bruvan sabhyo nāpnuyād dveṣakilbiṣe /
bruvāṇas tv anyathā sabhyas tad evobhayam āpnuyāt // Nar_M3.2 //
rājā tu dhārmikān sabhyān niyuñjyāt suparīkṣitān /
vyavahāradhuraṃ voḍhuṃ ye śaktāḥ sadgavā iva // Nar_M3.3 //
dharmaśāstrārthakuśalāḥ kulīnāḥ satyavādinaḥ /
samāḥ śatrau ca mitre ca nṛpateḥ syuḥ sabhāsadaḥ // Nar_M3.4 //
tatpratiṣṭhaḥ smṛto dharmo dharmamūlaś ca pārthivaḥ /
saha sadbhir ato rājā vyavahārān viśodhayet // Nar_M3.5 //
śuddheṣu vyavahāreṣu śuddhiṃ yānti sabhāsadaḥ /
śuddhiś ca teṣāṃ dharmād dhi dharmam eva vadet tataḥ // Nar_M3.6 //
yatra dharmo hy adharmeṇa satyaṃ yatrānṛtena ca /
hanyate prekṣamāṇānāṃ hatās tatra sabhāsadaḥ // Nar_M3.7 //
viddho dharmo hy adharmeṇa sabhāṃ yatropatiṣṭhate /
na ced viśalyaḥ kriyate viddhās tatra sabhāsadaḥ // Nar_M3.8 //
sabhā vā na praveṣṭavyā vaktavyaṃ vā samañjasam /
abruvan vibruvan vāpi naro bhavati kilbiṣī // Nar_M3.9 //
ye tu sabhyāḥ sabhāṃ gatvā tūṣṇīṃ dhyāyanta āsate /
yathāprāptaṃ na bruvate sarve te 'nṛtavādinaḥ // Nar_M3.10 //
pādo 'dharmasya kartāraṃ pādaḥ sākṣiṇam ṛcchati /
pādaḥ sabhāsadaḥ sarvān pādo rājānam ṛcchati // Nar_M3.11 //
rājā bhavaty anenās tu mucyante ca sabhāsadaḥ /
eno gacchati kartāraṃ nindārho yatra nindyate // Nar_M3.12 //
andho matsyān ivāśnāti nirapekṣaḥ sakaṇṭakān /
parokṣam arthavaikalyād bhāṣate yaḥ sabhāṃ gataḥ // Nar_M3.13 //
tasmāt sabhyaḥ sabhāṃ prāpya rāgadveṣavivarjitaḥ /
vacas tathāvidhaṃ brūyād yathā na narakaṃ patet // Nar_M3.14 //
yathā śalyaṃ bhiṣag vidvān uddhared yantrayuktitaḥ /
prāḍvivākas tathā śalyam uddhared vyavahārataḥ // Nar_M3.15 //
yatra sabhyo janaḥ sarvaḥ sādhv etad iti manyate /
sa niḥśalyo vivādaḥ syāt saśalyaḥ syād ato 'nyathā // Nar_M3.16 //
na sā sabhā yatra na santi vṛddhā vṛddhā na te ye na vadanti dharmam /
nāsau dharmo yatra na satyam asti na tat satyaṃ yac chalenānuviddham // Nar_M3.17 //
vyavahārapadāni
1. ṛṇādānam
ṛṇaṃ deyam adeyaṃ ca yena yatra yathā ca yat /
dānagrahaṇadharmāc ca ṛṇādānam iti smṛtam // Nar_1.01 //
pitary uparate putrā ṛṇaṃ dadyur yathāṃśataḥ /
vibhaktā hy avibhaktā vā yas tām udvahate dhuram // Nar_1.02 //
pitṛvyeṇāvibhaktena bhrātrā vā yad ṛṇaṃ kṛtam /
mātrā vā yat kuṭumbārthe dadyus tad rikthino 'khilam // Nar_1.03 //
kramād avyāhataṃ prāptaṃ putrair yan narṇam uddhṛtam /
dadyuḥ paitāmahaṃ pautrās tac caturthān nivartate // Nar_1.04 //
icchanti pitaraḥ putrān svārthahetor yatas tataḥ /
uttamarṇādhamarṇebhyo mām ayaṃ mocayiṣyati // Nar_1.05 //
ataḥ putreṇa jātena svārtham utsṛjya yatnataḥ /
pitā mokṣitavya ṛṇād yathā na narakaṃ patet // Nar_1.06 //
tapasvī cāgnihotrī ca ṛṇavān mriyate yadi /
tapaś caivāgnihotraṃ ca sarvaṃ tad dhanināṃ dhanam // Nar_1.07 //
na putrarṇaṃ pitā dadyād dadyāt putras tu paitṛkam /
kāmakrodhasurādyūta- prātibhāvyakṛtaṃ vinā // Nar_1.08 //
pitur eva niyogād yat kuṭumbabharaṇāya ca /
kṛtaṃ vā yad ṛṇaṃ kṛcchre dadyāt putrasya tat pitā // Nar_1.09 //
śiṣyāntevāsidāsastrī- vaiyāvṛttyakaraiś ca yat /
kuṭumbahetor utkṣiptaṃ voḍhavyaṃ tat kuṭumbinā // Nar_1.10 //
nārvāg viṃśatimād varṣāt pitari proṣite sutaḥ /
ṛṇaṃ dadyāt pitṛvye vā jyeṣṭhe bhrātary athāpi vā // Nar_1.11 //
dāpyaḥ pararṇam eko 'pi jīvatsv adhikṛtaiḥ kṛtam /
preteṣu tu na tatputraḥ pararṇaṃ dātum arhati // Nar_1.12 //
na strī patikṛtaṃ dadyād ṛṇaṃ putrakṛtaṃ tathā /
abhyupetād ṛte yadvā saha patyā kṛtaṃ bhavet // Nar_1.13 //
dadyād aputrā vidhavā niyuktā yā mumūrṣuṇā /
yo vā tadriktham ādadyād yato riktham ṛṇaṃ tataḥ // Nar_1.14 //
na ca bhāryākṛtam ṛṇaṃ kathaṃcit patyur ābhavet /
āpatkṛtād ṛte puṃsāṃ kuṭumbārtho hi vistaraḥ // Nar_1.15 //
anyatra rajakavyādha- gopaśauṇḍikayoṣitām /
teṣāṃ tatpratyayā vṛttiḥ kuṭumbaṃ ca tadāśrayam // Nar_1.16 //
putriṇī tu samutsṛjya putraṃ strī yānyam āśrayet /
ṛkthaṃ tasyā haret sarvaṃ niḥsvāyāḥ putra eva tu // Nar_1.17 //
yā tu sapradhanaiva strī sāpatyā cānyam āśrayet /
so 'syā dadyād ṛṇaṃ bhartur utsṛjed vā tathaiva tām // Nar_1.18 //
adhanasya hy aputrasya mṛtasyopaiti yaḥ striyam /
ṛṇaṃ voḍhuḥ sa bhajate tad evāsya dhanaṃ smṛtam // Nar_1.19 //
dhanastrīhāriputrāṇām ṛṇabhāg yo dhanaṃ haret /
putro 'satoḥ strīdhaninoḥ strīhārī dhaniputrayoḥ // Nar_1.20 //
uttamā svairiṇī yā syād uttamā ca punarbhuvām /
ṛṇaṃ tayoḥ patikṛtaṃ dadyād yas tām upāśnute // Nar_1.21 //
strīkṛtāny apramāṇāni kāryāṇy āhur anāpadi /
viśeṣato gṛhakṣetra- dānādhamanavikrayāḥ // Nar_1.22 //
etāny api pramāṇāni bhartā yady anumanyate /
putraḥ patyur abhāve vā rājā vā patiputrayoḥ // Nar_1.23 //
bhartrā prītena yad dattaṃ striyai tasmin mṛte 'pi tat /
sā yathākāmam aśnīyād dadyād vā sthāvarād ṛte // Nar_1.24 //
tathā dāsakṛtaṃ kāryam akṛtaṃ paricakṣate /
anyatra svāmisaṃdeśān na dāsaḥ prabhur ātmanaḥ // Nar_1.25 //
putreṇa ca kṛtaṃ kāryaṃ yat syāt pitur anicchataḥ /
tad apy akṛtam evāhur dāsaḥ putraś ca tau samau // Nar_1.26 //
aprāptavyavahāraś cet svatantro 'pi hi na rṇabhāk /
svātantryaṃ tu smṛtaṃ jyeṣṭhe jyaiṣṭhyaṃ guṇavayaḥkṛtam // Nar_1.27 //
trayaḥ svatantrā loke 'smin rājācāryas tathaiva ca /
prati prati ca varṇānāṃ sarveṣāṃ svagṛhe gṛhī // Nar_1.28 //
asvatantrāḥ prajāḥ sarvāḥ svatantraḥ pṛthivīpatiḥ /
asvatantraḥ smṛtaḥ śiṣya ācārye tu svatantratā // Nar_1.29 //
asvatantrāḥ striyaḥ putrā dāsāś ca saparigrahāḥ /
svatantras tatra tu gṛhī yasya yat syāt kramāgatam // Nar_1.30 //
garbhasthaiḥ sadṛśo jñeya ā varṣād aṣṭamāc chiṣuḥ /
bāla ā ṣoḍaśāj jñeyaḥ pogaṇḍaś cāpi śabdyate // Nar_1.31 //
parato vyavahārajñaḥ svatantraḥ pitarau vinā /
jīvator asvatantraḥ syāj jarayāpi samanvitaḥ // Nar_1.32 //
tayor api pitā śreyān bījaprādhānyadarśanāt /
abhāve bījino mātā tadabhāve tu pūrvajaḥ // Nar_1.33 //
svatantrāḥ sarva evaite paratantreṣu sarvadā /
anuśiṣṭau visarge ca vikraye ceśvarā matāḥ // Nar_1.34 //
yad bālaḥ kurute kāryam asvatantras tathaiva ca /
akṛtaṃ tad iti prāhuḥ śāstre śāstravido janāḥ // Nar_1.35 //
svatantro 'pi hi yat kāryaṃ kuryād aprakṛtiṃ gataḥ /
tad apy akṛtam evāhur asvatantraḥ sa hetutaḥ // Nar_1.36 //
kāmakrodhābhiyuktārta- bhayavyasanapīḍitāḥ /
rāgadveṣaparītāś ca jñeyās tv aprakṛtiṃ gatāḥ // Nar_1.37 //
kule jyeṣṭhas tathā śreṣṭhaḥ prakṛtisthaś ca yo bhavet /
tatkṛtaṃ syāt kṛtaṃ kāryaṃ nāsvatantrakṛtaṃ kṛtam // Nar_1.38 //
dhanamūlāḥ kriyāḥ sarvā yatnas tatsādhane mataḥ /
rakṣaṇaṃ vardhanaṃ bhoga iti tasya vidhiḥ kramāt // Nar_1.39 //
tat punas trividhaṃ jñeyaṃ śuklaṃ śabalam eva ca /
kṛṣṇaṃ ca tasya vijñeyaḥ prabhedaḥ saptadhā pṛthak // Nar_1.40 //
śrutaśauryatapaḥkanyā- śiṣyayājyānvayāgatam /
dhanaṃ saptavidhaṃ śuklam udayo 'py asya tadvidhaḥ // Nar_1.41 //
kusīdakṛṣivāṇijya- śulkaśilpānuvṛttibhiḥ /
kṛtopakārād āptaṃ ca śabalaṃ samudāhṛtam // Nar_1.42 //
pārśvikadyūtadautyārti- pratirūpakasāhasaiḥ /
vyājenopārjitaṃ yac ca tat kṛṣṇaṃ samudāhṛtam // Nar_1.43 //
tena krayo vikrayaś ca dānaṃ grahaṇam eva ca /
vividhāś ca pravartante kriyāḥ saṃbhoga eva ca // Nar_1.44 //
yathāvidhena dravyeṇa yat kiṃcit kurute naraḥ /
tathāvidham avāpnoti sa phalaṃ pretya ceha ca // Nar_1.45 //
tat punar dvādaśavidhaṃ prativarṇāśrayāt smṛtam /
sādhāraṇaṃ syāt trividhaṃ śeṣaṃ navavidhaṃ smṛtam // Nar_1.46 //
kramāgataṃ prītidāyaḥ prāptaṃ ca saha bhāryayā /
aviśeṣeṇa varṇānāṃ sarveṣāṃ trividhaṃ dhanam // Nar_1.47 //
vaiśeṣikaṃ dhanaṃ jñeyaṃ brāhmaṇasya trilakṣaṇam /
pratigraheṇa yallabdhaṃ yājyataḥ śiṣyatas tathā // Nar_1.48 //
trividhaṃ kṣatriyasyāpi prāhur vaiśeṣikaṃ dhanam /
yuddhopalabdhaṃ kāraś ca daṇḍaś ca vyavahārataḥ // Nar_1.49 //
vaiśeṣikaṃ dhanaṃ jñeyaṃ vaiśyasyāpi trilakṣaṇam /
kṛṣigorakṣavāṇijyaiḥ śūdrasyaibhyas tv anugrahāt // Nar_1.50 //
sarveṣām eva varṇānām eṣa dharmyo dhanāgamaḥ /
viparyayād adharmyaḥ syān na ced āpad garīyasī // Nar_1.51 //
āpatsv anantarā vṛttir brāhmaṇasya vidhīyate /
vaiśyavṛttis tataś coktā na jaghanyā kathaṃcana // Nar_1.52 //
na kathaṃcana kurvīta brāhmaṇaḥ karma vārṣalam /
vṛṣalaḥ karma na brāhmaṃ patanīye hi te tayoḥ // Nar_1.53 //
utkṛṣṭaṃ cāpakṛṣṭaṃ ca tayoḥ karma na vidyate /
madhyame karmaṇī hitvā sarvasādhāraṇe hi te // Nar_1.54 //
āpadaṃ brāhmaṇas tīrtvā kṣatravṛttyā hṛtair dhanaiḥ /
utsṛjet kṣatravṛttiṃ tāṃ kṛtvā pāvanam ātmanaḥ // Nar_1.55 //
tasyām eva tu yo vṛttau brāhmaṇo ramate rasāt /
kāṇḍapṛṣṭhaś cyuto mārgāt so 'pāṅkteyaḥ prakīrtitaḥ // Nar_1.56 //
vaiśyavṛttāv avikreyaṃ brāhmaṇasya payo dadhi /
ghṛtaṃ madhu madhūcchiṣṭaṃ lākṣākṣārarasāsavāḥ // Nar_1.57 //
māṃsaudanatilakṣauma- somapuṣpaphalapalāḥ /
manuṣyaviṣaśastrāmbu- lavaṇāpūpavīrudhaḥ // Nar_1.58 //
nīlīkauṣeyacarmāsthi- kutapaikaśaphā mṛdaḥ /
udaśvitkeśapiṇyāka- śākādyauṣadhayas tathā // Nar_1.59 //
brāhmaṇasya tu vikreyaṃ śuṣkaṃ dāru tṛṇāni ca /
gandhadravyairakāvetra- tūlamūlatuśād ṛte // Nar_1.60 //
svayaṃ śīrṇaṃ ca vidalaṃ phalānāṃ badareṅgude /
rajjuḥ kārpāsikaṃ sūtraṃ tac ced avikṛtaṃ bhavet // Nar_1.61 //
aśaktau bheṣajasyārthe yajñahetos tathaiva ca /
yady avaśyaṃ tu vikreyās tilā dhānyena tatsāmāḥ // Nar_1.62 //
avikreyāṇi vikrīṇan brāhmaṇaḥ pracyutaḥ pathaḥ /
mārge punar avasthāpya rājñā daṇḍena bhūyasā // Nar_1.63 //
pramāṇāni pramāṇasthaiḥ paripālyāni yatnataḥ /
sīdanti hi pramāṇāni pramāṇair avyavasthitaiḥ // Nar_1.64 //
likhitaṃ sākṣiṇo bhuktiḥ pramāṇaṃ trividhaṃ smṛtaṃ /
dhanasvīkaraṇe yena dhanī dhanam upāśnute // Nar_1.65 //
likhitaṃ balavan nityaṃ jīvantas tv eva sākṣiṇaḥ /
kālātiharaṇād bhuktir iti śāstreṣu niścayaḥ // Nar_1.66 //
trividhasyāsya dṛṣṭasya pramāṇasya yathākramam /
pūrvaṃ pūrvaṃ guru jñeyaṃ bhuktir ebhyo garīyasī // Nar_1.67 //
vidyamāne 'pi likhite jīvatsv api hi sākṣiṣu /
viśeṣataḥ sthāvarāṇāṃ yan na bhuktaṃ na tat sthiram // Nar_1.68 //
bhujyamānān parair arthān yaḥ svān maurkhyād upekṣate /
samakṣaṃ jīvato 'py asya tān bhuktiḥ kurute vaśe // Nar_1.69 //
yatkiṃcid daśa varṣāṇi saṃnidhau prekṣate dhanī /
bhujyamānaṃ parais tūṣṇīṃ na sa tal labdhum arhati // Nar_1.70 //
upekṣāṃ kurvatas tasya tūṣṇīṃ bhūtasya tiṣṭhataḥ /
kāle 'tipanne pūrvokte vyavahāro na sidhyati // Nar_1.71 //
ajaḍaś ced apogaṇḍo viṣaye cāsya bhujyate /
bhuktaṃ tad vyavahāreṇa bhoktā tad dhanam arhati // Nar_1.72 //
ādhiḥ sīmā bāladhanaṃ nikṣepopanidhī striyaḥ /
rājasvaṃ śrotriyasvaṃ ca nopabhogena jīryate // Nar_1.73 //
pratyakṣaparibhogāc ca svāmino dvidaśāḥ samāḥ /
ādhyādīny api jīryante strīnarendradhanād ṛte // Nar_1.74 //
strīdhanaṃ ca narendrāṇāṃ na kadācana jīryate /
anāgamaṃ bhujyamānaṃ vatsarāṇāṃ śatair api // Nar_1.75 //
nirbhogo yatra dṛśyeta na dṛśyetāgamaḥ kvacit /
āgamaḥ kāraṇaṃ tatra na bhogas tatra kāraṇam // Nar_1.76 //
anāgamaṃ bhujyate yan na tad bhogo 'tivartate /
prete tu bhoktari dhanaṃ yāti tadvaṃśyabhogyatām // Nar_1.77 //
āhartaivābhiyuktaḥ sann arthānām uddharet padam /
bhuktir eva viśuddhiḥ syāt prāptānāṃ pitṛtaḥ kramāt // Nar_1.78 //
anvāhitaṃ hṛtaṃ nyastaṃ balāvaṣṭabdhaṃ yācitam /
apratyakṣaṃ ca yad bhuktaṃ ṣaḍ etāny āgamaṃ vinā // Nar_1.79 //
tathārūḍhavivādasya pretasya vyavahāriṇaḥ /
putreṇa so 'rthaḥ saṃśodhyo na taṃ bhogo 'tivartate // Nar_1.80 //
yad vināgamam apy ūrdhvaṃ bhuktaṃ pūrvais tribhir bhavet /
na tac chakyam apākartuṃ kramāt tripuruṣāgatam // Nar_1.81 //
santo 'pi na pramāṇaṃ syur mṛte dhanini sākṣiṇaḥ /
anyatra śrāvitaṃ yat syāt svayam āsannamṛtyunā // Nar_1.82 //
na hi pratyarthini prete pramāṇaṃ sākṣiṇāṃ vacaḥ /
sākṣimat karaṇaṃ tatra pramāṇaṃ syād viniścaye // Nar_1.83 //
śrāvitas tv ātureṇāpi yas tv artho dharmasaṃhitaḥ /
mṛte 'pi tatra sākṣī syāt ṣaṭsu cānvāhitādiṣu // Nar_1.84 //
kriya rṇādiṣu sarveṣu balavaty uttarottarā /
pratigrahādhikrīteṣu pūrvā pūrvā garīyasī // Nar_1.85 //
sthānalābhanimittaṃ hi dānagrahaṇam iṣyate /
tat kusīdam iti proktaṃ tena vṛttiḥ kusīdinām // Nar_1.86 //
kāyikā kālikā caiva kāritā ca tathā smṛtā /
cakravṛddhiś ca śāstreṣu tasya vṛddhiś caturvidhā // Nar_1.87 //
kāyāvirodhinī śaśvat paṇapādyā tu kāyikā /
pratimāsaṃ sravati yā vṛddhiḥ sā kālikā smṛtā // Nar_1.88 //
vṛddhiḥ sā kāritā nāma ya rṇikena svayaṃkṛtā /
vṛddher api punar vṛddhiś cakravṛddhir udāhṛtā // Nar_1.89 //
ṛṇānāṃ sārvabhaumo 'yaṃ vidhir vṛddhikaraḥ smṛtaḥ /
deśācāravidhis tv anyo yatra rṇam avatiṣṭhati // Nar_1.90 //
dviguṇaṃ triguṇaṃ caiva tathānyasmiṃś caturguṇam /
tathāṣṭaguṇam anyasmin deśe deśe 'vatiṣṭhate // Nar_1.91 //
hiraṇyadhānyavastrāṇāṃ vṛddhir dvitricaturguṇā /
ghṛtasyāṣṭaguṇā vṛddhiḥ strīpaśūnāṃ ca saṃtatiḥ // Nar_1.92 //
sūtrakarpāsakiṇvānāṃ trapuṣaḥ sīsakasya ca /
āyudhānāṃ ca sarveṣāṃ carmaṇas tāmralohayoḥ // Nar_1.93 //
anyeṣāṃ caiva sarveṣām iṣṭakānāṃ tathaiva ca /
akṣayyā vṛddhir eteṣāṃ manur āha prajāpatiḥ // Nar_1.94 //
tailānāṃ caiva sarveṣāṃ madyānāṃ madhusarpiṣām /
vṛddhir aṣṭaguṇā jñeyā guḍasya lavaṇasya ca // Nar_1.95 //
na vṛddhiḥ prītidattānāṃ syād anākāritā kvacit /
anākāritam apy ūrdhvaṃ vatsarārdhād vivardhate // Nar_1.96 //
eṣa vṛddhividhiḥ proktaḥ prativṛddhasya dharmataḥ /
vṛddhis tu yoktā dhānyānāṃ vārdhuṣyaṃ tad udāhṛtam // Nar_1.97 //
āpadaṃ nistared vaiśyaḥ kāmaṃ vārdhuṣakarmaṇā /
āpatsv api hi kaṣṭāsu brāhmaṇasya na vārdhuṣam // Nar_1.98 //
brāhmaṇasya tu yad deyaṃ sānvayasya na cāsti saḥ /
svakulyasyāsya nivapet tadabhāve 'sya bandhuṣu // Nar_1.99 //
yadā tu na svakulyāḥ syur na ca saṃbandhibāndhavāḥ /
tadā dadyāt svajātibhyas teṣv asatsv apsu nikṣipet // Nar_1.100 //
gṛhītvopagataṃ dadyād ṛṇikāyodayaṃ dhanī /
adadad yācyamānas tu śeṣahānim avāpnuyāt // Nar_1.101 //
lekhyaṃ dadyād ṛṇe śuddhe tadabhāve pratiśravam /
dhanikarṇikayor evaṃ viśuddhiḥ syāt parasparam // Nar_1.102 //
viśrambhahetū dvāv atra pratibhūr ādhir eva ca /
likhitaṃ sākṣiṇaś ca dve pramāṇe vyaktikārake // Nar_1.103 //
upasthānāya dānāya pratyayāya tathaiva ca /
trividhaḥ pratibhūr dṛṣṭas triṣv evārtheṣu sūribhiḥ // Nar_1.104 //
ṛṇiṣv apratikurvatsu pratyaye vā vivādite /
pratibhūs tad ṛṇaṃ dadyād anupasthāpayaṃs tathā // Nar_1.105 //
bahavaś cet pratibhuvo dadyus te 'rthaṃ yathākṛtam /
arthe 'viśeṣite hy eṣu dhaninaś chandataḥ kriyā // Nar_1.106 //
yaṃ cārthaṃ pratibhūr dadyād dhanikenopapīḍitaḥ /
ṛṇikas taṃ pratibhuve dviguṇaṃ pratipādayet // Nar_1.107 //
adhikriyata ity ādhiḥ sa vijñeyo dvilakṣaṇaḥ /
kṛtakālopaneyaś ca yāvaddeyodyatas tathā // Nar_1.108 //
sa punar dvividhaḥ prokto gopyo bhogyas tathaiva ca /
pratidānaṃ tathaivāsya lābhahānir viparyaye // Nar_1.109 //
pramādād dhaninas tadvad ādhau vikṛtim āgate /
vinaṣṭe mūlanāśaḥ syād daivarājakṛtād ṛte // Nar_1.110 //
rakṣyamāṇo 'pi yatrādhiḥ kāleneyād asāratām /
ādhir anyo 'dhikartavyo deyaṃ vā dhanine dhanam // Nar_1.111 //
atha śaktivihīnaḥ syād ṛṇī kālaviparyayāt /
śakyaprekṣam ṛṇaṃ dāpyaḥ kāle kāle yathodayam // Nar_1.112 //
śakto vā yadi daurātmyān na dadyād dhanine dhanam /
rājñā dāpayitavyaḥ syād gṛhītvāṃśaṃ tu viṃśakam // Nar_1.113 //
naśyed ṛṇaparīmāṇaṃ kāleneha rṇikasya cet /
jātisaṃjñādhivāsānām āgamo lekhyataḥ smṛtaḥ // Nar_1.114 //
lekhyaṃ tu dvividhaṃ jñeyaṃ svahastānyakṛtaṃ tathā /
asākṣimat sākṣimac ca siddhir deśasthites tayoḥ // Nar_1.115 //
deśācārāviruddhaṃ yad vyaktādhikṛtalakṣaṇam /
tat pramāṇaṃ smṛtaṃ lekhyam aviluptakramākṣaram // Nar_1.116 //
mattābhiyuktastrībāla- balātkārakṛtaṃ ca yat /
tad apramāṇakaraṇaṃ bhītopadhikṛtaṃ tathā // Nar_1.117 //
mṛtāḥ syuḥ sākṣiṇo yatra dhanikarṇikalekhakāḥ /
tad apy apārthaṃ likhitam ṛte tv ādheḥ sthirāśrayāt // Nar_1.118 //
ādhir yo dvividhaḥ prokto jaṅgamaḥ sthāvaras tathā /
siddhir atrobhayasyāsya bhogo yady asti nānyathā // Nar_1.119 //
darśitaṃ pratikālaṃ yac chrāvitaṃ śrāvitaṃ ca yat /
lekhyaṃ sidhyati sarvatra mṛteṣv api hi sākṣiṣu // Nar_1.120 //
aśrutārtham adṛṣṭārthaṃ vyavahārārtham eva ca /
na lekhyaṃ siddhim āpnoti jīvatsv api hi sākṣiṣu // Nar_1.121 //
lekhye deśāntaranyaste dagdhe durlikhite hṛte /
satas tatkālakaraṇam asato dṛṣṭadarśanam // Nar_1.122 //
yasmin syāt saṃśayo lekhye bhūtābhūtakṛte kvacit /
tatsvahastakriyācihna- prāptiyuktibhir uddharet // Nar_1.123 //
lekhyaṃ yac cānyanāmāṅkaṃ hetvantarakṛtaṃ bhavet /
vipratyaye parīkṣyaṃ tat saṃbandhāgamahetubhiḥ // Nar_1.124 //
lekhyaṃ yac cānyanāmāṅkaṃ hetvantarakṛtaṃ bhavet /
vipratyaye parīkṣyaṃ tat saṃbandhāgamahetubhiḥ // Nar_1.124*1 //
likhitaṃ likhitenaiva sākṣimat sākṣibhir haret /
sākṣibhyo likhitaṃ śreyo likhitena tu sākṣiṇaḥ // Nar_1.125 //
chinnabhinnahṛtonmṛṣṭa- naṣṭadurlikhiteṣu ca /
kartavyam anyal lekhyaṃ syād eṣa lekhyavidhiḥ smṛtaḥ // Nar_1.126 //
saṃdigdheṣu tu kāryeṣu dvayor vivadamānayoḥ /
dṛṣṭaśrutānubhūtatvāt sākṣibhyo vyaktidarśanam // Nar_1.127 //
samakṣadarśanāt sākṣī vijñeyaḥ śrotracakṣuṣoḥ /
śrotrasya yat paro brūte cakṣuṣaḥ kāyakarma yat // Nar_1.128 //
ekādaśavidhaḥ sākṣī sa tu dṛṣṭo manīṣibhiḥ /
kṛtaḥ pañcavidhas teṣāṃ ṣaḍvidho 'kṛta ucyate // Nar_1.129 //
likhitaḥ smāritaś caiva yadṛcchābhijña eva ca /
gūḍhaś cottarasākṣī ca sākṣī pañcavidhaḥ smṛtaḥ // Nar_1.130 //
akṛtaḥ ṣaḍvidhaś cāpi sūribhiḥ parikīrtitaḥ /
grāmaś ca prāḍvivākaś ca rājā ca vyavahāriṇām // Nar_1.131 //
kāryeṣv abhyantaro yaḥ syād arthinā prahitaś ca yaḥ /
kulaṃ kulavivādeṣu bhaveyus te 'pi sākṣiṇaḥ // Nar_1.132 //
kulīnā ṛjavaḥ śuddhā janmataḥ karmato 'rthataḥ /
tryavarāḥ sākṣiṇo 'nindyāḥ śucayaḥ syuḥ subuddhayaḥ // Nar_1.133 //
brāhmaṇāḥ kṣatriyā vaiśyāḥ śūdrā ye cāpy aninditāḥ /
prativarṇaṃ bhaveyus te sarve sarveṣu vā punaḥ // Nar_1.134 //
śreṇīṣu śreṇipuruṣāḥ sveṣu vargeṣu vargiṇaḥ /
bahirvāsiṣu bāhyāś ca striyaḥ striṣu ca sākṣiṇaḥ // Nar_1.135 //
śreṇyādiṣu tu vargeṣu kaścic ced dveṣyatām iyāt /
tebhya eva na sākṣī syād dveṣṭāraḥ sarva eva te // Nar_1.136 //
asākṣy api hi śāstreṣu dṛṣṭaḥ pañcavidho budhaiḥ /
vacanād doṣato bhedāt svayamukter mṛtāntaraḥ // Nar_1.137 //
śrotriyādyā vacanataḥ stenādyā doṣadarśanāt /
bhedād vipratipattiḥ syād vivāde yatra sākṣiṇaḥ // Nar_1.138 //
svayamukter anirdiṣṭaḥ svayam evaitya yo vadet /
mṛtāntaro 'rthini prete mumūrṣuśrāvitād ṛte // Nar_1.139 //
śrotriyās tāpasā vṛddhā ye ca pravrajitā narāḥ /
asākṣiṇas te vacanān nātra hetur udāhṛtaḥ // Nar_1.140 //
stenāḥ sāhasikāś caṇḍāḥ kitavā vadhakās tathā /
asākṣiṇas te duṣṭatvāt teṣu satyaṃ na vidyate // Nar_1.141 //
rājñā parigṛhīteṣu sākṣiṣv ekārthaniścaye /
vacanaṃ yatra bhidyate te syur bhedād asākṣiṇaḥ // Nar_1.142 //
anirdiṣṭas tu sākṣitve svayam evaitya yo vadet /
sūcīty uktaḥ sa śāstreṣu na sa sākṣitvam arhati // Nar_1.143 //
yo 'rthaḥ śrāvayitavyaḥ syāt tasminn asati cārthini /
kva tad vadatu sākṣitvam ity asākṣī mṛtāntaraḥ // Nar_1.144 //
yo 'rthaḥ śrāvayitavyaḥ syāt tasminn asati cārthini /
kva tad vadatu sākṣitvam ity asākṣī mṛtāntaraḥ // Nar_1.144*1 //
dvayor vivadator arthe dvayoḥ satsu ca sākṣiṣu /
pūrvapakṣo bhaved yasya bhaveyus tasya sākṣiṇaḥ // Nar_1.145 //
ādharyaṃ pūrvapakṣasya yasminn arthe vaśād bhavet /
praṣṭavyāḥ sākṣiṇas tatra vivāde prativādinaḥ // Nar_1.146 //
na pareṇa samuddiṣṭam upeyāt sākṣiṇaṃ rahaḥ /
bhedayet taṃ na cānyena hīyetaivaṃ samācaran // Nar_1.147 //
sākṣy uddiṣṭo yadi preyād gacched vāpi digantaram /
tacchrotāraḥ pramāṇaṃ syuḥ pramāṇaṃ hy uttarakriyā // Nar_1.148 //
sudīrgheṇāpi kālena likhitaṃ siddhim āpnuyāt /
jānatā cātmanā lekhyaṃ ajānānas tu lekhayet // Nar_1.149 //
siddhir uktāṣṭamād varṣāt smāritasyeha sākṣiṇaḥ /
ā pañcamāt tathā siddhir yadṛcchopagatasya tu // Nar_1.150 //
ā tṛtīyāt tathā varṣāt siddhir gūḍhasya sākṣiṇaḥ /
ā vai saṃvatsarāt siddhiṃ vadanty uttarasākṣiṇaḥ // Nar_1.151 //
athavā kālaniyamo na dṛṣṭaḥ sākṣiṇaṃ prati /
smṛtyapekṣaṃ hi sākṣitvam āhuḥ śāstravido janāḥ // Nar_1.152 //
yasya nopahatā puṃsaḥ smṛtiḥ śrotraṃ ca nityaśaḥ /
sudīrgheṇāpi kālena sa sākṣī sākṣyam arhati // Nar_1.153 //
asākṣipratyayās tv anye ṣaḍvivādāḥ prakīrtitāḥ /
lakṣaṇāny eva sākṣitvaṃ eṣām āhur manīṣiṇaḥ // Nar_1.154 //
ulkāhasto 'gnido jñeyaḥ śastrapāṇis tu ghātakaḥ /
keśākeśigṛhītaś ca yugapat pāradārikaḥ // Nar_1.155 //
kuddālapāṇir vijñeyaḥ setubhettā samīpagaḥ /
tathā kuṭhārapāṇiś ca vanachettā prakīrtitaḥ // Nar_1.156 //
abhyagracihno vijñeyo daṇḍapāruṣyakṛn naraḥ /
asākṣipratyayā hy ete pāruṣye tu parīkṣaṇam // Nar_1.157 //
kaścit kṛtvātmanaś cihnaṃ dveṣāt param upadravet /
hetvarthagatisāmarthyais tatra yuktaṃ parīkṣaṇam // Nar_1.158 //
nārthasaṃbandhino nāptā na sahāyā na vairiṇaḥ /
na dṛṣṭadoṣāḥ praṣṭavyāś na vyādhyārtā na dūṣitāḥ // Nar_1.159 //
dāsanaikṛtikāśraddha- vṛddhastrībālacākrikāḥ /
mattonmattapramattārta- kitavagrāmayājakāḥ // Nar_1.160 //
mahāpathikasāmudra- vaṇikpravrajitāturāḥ /
lubdhakaśrotriyācāra- hīnaklībakuśīlavāḥ // Nar_1.161 //
nāstikavrātyadārāgni- tyāgino 'yājyayājakāḥ /
ekasthālīsahāyāri- carajñātisanābhayaḥ // Nar_1.162 //
prāgdṛṣṭadoṣaśailūṣa viṣajīvyahituṇḍikāḥ /
garadāgnidakīnāśa- śūdrāputropapātikāḥ // Nar_1.163 //
klāntasāhasikaśrānta- nirdhanāntyāvasāyinaḥ /
bhinnavṛttāsamāvṛtta- jaḍatailikamūlikāḥ // Nar_1.164 //
bhūtāviṣṭanṛpadviṣṭa varṣanakṣatrasūcakāḥ /
aghaśaṃsyātmavikretṛ- hīnāṅgabhagavṛttayaḥ // Nar_1.165 //
kunakhī śyāvadan śvitri- mitradhrukśaṭhaśauṇḍikāḥ /
aindrajālikalubdhogra śreṇīgaṇavirodhinaḥ // Nar_1.166 //
vadhakṛccitrakṛnmaṅkhaḥ patitaḥ kūṭakārakaḥ /
kuhakaḥ pratyavasitas taskaro rājapūruṣaḥ // Nar_1.167 //
manuṣyaviṣaśastrāmbu- lavaṇāpūpavīrudhām /
vikretā brāhmaṇaś caiva dvijo vārdhuṣikaś ca yaḥ // Nar_1.168 //
cyutaḥ svadharmāt kulikaḥ stāvako hīnasevakaḥ /
pitrā vivadamānaś ca bhedakṛc cety asākṣiṇaḥ // Nar_1.169 //
asākṣiṇo ye nirdiṣṭā dāsanaikṛtikādayaḥ /
kāryagauravam āsādya bhaveyus te 'pi sākṣiṇaḥ // Nar_1.170 //
sāhaseṣu ca sarveṣu steyasaṃgrahaṇeṣu ca /
pāruṣyayoś cāpy ubhayor na parīkṣeta sākṣiṇaḥ // Nar_1.171 //
teṣām api na bālaḥ syān naiko na strī na kūṭakṛt /
na bāndhavo na cārātir brūyus te sākṣyam anyathā // Nar_1.172 //
bālo 'jñānād asatyāt strī pāpābhyāsāc ca kūṭakṛt /
vibrūyād bāndhavaḥ snehād vairaniryātanād ariḥ // Nar_1.173 //
athavānumato yaḥ syād dvayor vivadamānayoḥ /
asākṣy eko 'pi sākṣitve praṣṭavyaḥ syāt sa saṃsadi // Nar_1.174 //
yas tv ātmadoṣabhinnatvād asvastha iva lakṣyate /
sthānāt sthānāntaraṃ gacched ekaikaṃ copadhāvati // Nar_1.175 //
kāsate 'nibhṛto 'kasmād abhīkṣṇaṃ niśvasaty api /
bhūmiṃ likhati pādābhyāṃ bāhu vāso dhunoti ca // Nar_1.176 //
bhidyate mukhavarṇo 'sya lalāṭaṃ svidyate tathā /
śoṣam āgacchataś coṣṭhāv ūrdhvaṃ tiryak ca vīkṣate // Nar_1.177 //
tvaramāṇa ivābaddham apṛṣṭo bahu bhāṣate /
kūṭasākṣī sa vijñeyas taṃ pāpaṃ vinayen nṛpaḥ // Nar_1.178 //
śrāvayitvā ca yo 'nyebhyaḥ sākṣitvaṃ tad vinihnute /
sa vineyo bhṛśataraṃ kūṭasākṣyadhiko hi saḥ // Nar_1.179 //
āhūya sākṣiṇaḥ pṛcchen niyamya śapathair bhṛśam /
samastān viditācārān vijñātārthān pṛthak pṛthak // Nar_1.180 //
satyena śāpayed vipraṃ kṣatriyaṃ vāhanāyudhaiḥ /
gobījakāñcanair vaiśyaṃ śūdraṃ sarvais tu pātakaiḥ // Nar_1.181 //
purāṇair dharmavacanaiḥ satyamāhātmyakīrtanaiḥ /
anṛtasyāpavādaiś ca bhṛśam uttrāsya sākṣiṇaḥ // Nar_1.182 //
nagno muṇḍaḥ kapālena bhikṣārthī kṣutpipāsitaḥ /
dīnaḥ śatrugṛhaṃ gacched yaḥ sākṣyam anṛtaṃ vadet // Nar_1.183 //
nagare pratiruddhaḥ san bahirdvāre bubhukṣitaḥ /
amitrān bhūyaśaḥ paśyed yaḥ sākṣyam anṛtaṃ vadet // Nar_1.184 //
yāṃ rātrim adhivinnā strī yāṃ caivākṣaparājitaḥ /
yāṃ ca bhārābhitaptāṅgo durvivaktā sa tāṃ vaset // Nar_1.185 //
sākṣī sākṣyasamuddeśe gokarṇaśithilaṃ caran /
sahasraṃ vāruṇān pāśān ātmani pratimuñcati // Nar_1.186 //
tasya varṣaśate pūrṇe pāśam ekaṃ pramucyate /
evaṃ sa bandhanāt tasmān mucyate niyutāḥ samāḥ // Nar_1.187 //
yāvato bāndhavāṃs yasmin hanti sākṣye 'nṛtaṃ vadan /
tāvataḥ saṃkhyayā tasmin śṛṇu saumyānupūrvaśaḥ // Nar_1.188 //
pañca paśvanṛte hanti daśa hanti gavānṛte /
śatam aśvānṛte hanti sahasraṃ puruṣānṛte // Nar_1.189 //
hanti jātān ajātāṃś ca hiraṇyārthe 'nṛtaṃ vadan /
sarvaṃ bhūmyanṛte hanti mā sma bhūmyanṛtaṃ vadīḥ // Nar_1.190 //
ekam evādvitīyaṃ tat prāhuḥ pāvanam ātmanaḥ /
satyaṃ svargasya sopānaṃ pārāvārasya naur iva // Nar_1.191 //
aśvamedhasahasraṃ ca satyaṃ ca tulayā dhṛtam /
aśvamedhasahasrād dhi satyam eva viśiṣyate // Nar_1.192 //
varaṃ kūpaśatād vāpi varaṃ vāpīśatāt kratuḥ /
varaṃ kratuśatāt putraḥ satyaṃ putraśatād varam // Nar_1.193 //
bhūr dhārayati satyena satyenodeti bhāskaraḥ /
satyena vāyuḥ pavate satyenāpaḥ sravanti ca // Nar_1.194 //
satyam eva paraṃ dānaṃ satyam eva paraṃ tapaḥ /
satyam eva paro dharmo lokānām iti naḥ śrutam // Nar_1.195 //
satyaṃ devāḥ samāsena manuṣyās tv anṛtaṃ smṛtam /
ihaiva tasya devatvaṃ yasya satye sthitā matiḥ // Nar_1.196 //
satyaṃ brūhy anṛtaṃ tyaktvā satyena svargam eṣyasi /
uktvānṛtaṃ mahāghoraṃ narakaṃ pratipatsyate // Nar_1.197 //
nirayeṣu ca te śaśvaj jihvām utkṛtya dāruṇāḥ /
asibhiḥ śātayiṣyanti balino yamakiṃkarāḥ // Nar_1.198 //
śūlair bhetsyanti cākruddhāḥ krośantam aparāyaṇam /
avākśirasam utkṣipya kṣepsyanty agnihradeṣu ca // Nar_1.199 //
anubhūya ca duḥkhās tāś ciraṃ narakavedanāḥ /
iha yāsyasy abhavyāsu gṛdhrakākādiyoniṣu // Nar_1.200 //
jñātvaitān anṛte doṣāñ jñātvā satye ca sadguṇān /
satyaṃ vadoddharātmānaṃ mātmānaṃ pātayiṣyasi // Nar_1.201 //
na bāndhavā na suhṛdo na dhanāni mahānty api /
alaṃ tārayituṃ śaktās tamasy ugre nimajjataḥ // Nar_1.202 //
pitaras tv avalambante tvayi sākṣitvam āgate /
tārayiṣyati kiṃvāsmān ātmānaṃ pātayiṣyati // Nar_1.203 //
satyam ātmā manuṣyasya satye sarvaṃ pratiṣṭhitam /
sarvathaivātmanātmānaṃ śreyasā yojayiṣyasi // Nar_1.204 //
yāṃ ca rātrim ajaniṣṭhā yāṃ rātriṃ ca mariṣyasi /
vṛthā tadantaraṃ te syāt kuryāś cet satyam anyathā // Nar_1.205 //
nāsti satyāt paro dharmo nānṛtāt pātakaṃ param /
sākṣidharme viśeṣeṇa satyam eva vadet tataḥ // Nar_1.206 //
yaḥ parārthe praharati svāṃ vācaṃ puruṣādhamaḥ /
ātmārthe kiṃ na kuryāt sa pāpo narakanirbhayaḥ // Nar_1.207 //
arthā vai vāci niyatā vāṅmūlā vāgviniḥsṛtāḥ /
yo hy etāṃ stenayed vācaṃ sa sarvasteyakṛn naraḥ // Nar_1.208 //
sākṣivipratipattau tu pramāṇaṃ bahavo yataḥ /
tatsāmye śucayo grāhyās tatsāmye smṛtimattarāḥ // Nar_1.209 //
smṛtimatsākṣisāmyaṃ tu vivāde yatra dṛśyate /
sūkṣmatvāt sākṣidharmasya sākṣyaṃ vyāvartate punaḥ // Nar_1.210 //
nirdiṣṭeṣv arthajāteṣu sākṣī cet sākṣyam āgataḥ /
na brūyād akṣarasamaṃ na tan nigaditaṃ bhavet // Nar_1.211 //
deśakālavayodravya- pramāṇākṛtijātiṣu /
yatra vipratipattiḥ syāt sākṣyaṃ tad asad ucyate // Nar_1.212 //
ūnam abhyadhikaṃ cārthaṃ prabrūyur yatra sākṣiṇaḥ /
tad apy anuktaṃ vijñeyam eṣa sākṣyavidhiḥ smṛtaḥ // Nar_1.213 //
pramādād dhanino yatra na syāl lekhyaṃ na sākṣiṇaḥ /
arthaṃ cāpahnuyād vādī tatroktas trividho vidhiḥ // Nar_1.214 //
codanā pratikālaṃ ca yuktileśas tathaiva ca /
tṛtīyaḥ śapathaś coktas tair evaṃ sādhayet kramāt // Nar_1.215 //
abhīkṣṇaṃ codyamāno yaḥ pratihanyān na tadvacaḥ /
tricatuḥpañcakṛtvo vā parato 'rthaṃ tam āvahet // Nar_1.216 //
codanāpratighāte tu yuktileśais tam anviyāt /
deśakālārthasaṃbandha- parimāṇakriyādibhiḥ // Nar_1.217 //
yuktiṣv apy asamarthāsu śapathair enam ardayet /
arthakālabalāpekṣam agnyambusukṛtādibhiḥ // Nar_1.218 //
dīptāgnir yaṃ na dahati yam antardhārayanty āpaḥ /
sa taraty abhiśāpaṃ taṃ kilbiṣī syād viparyaye // Nar_1.219 //
strīṇāṃ śīlābhiyogeṣu steyasāhasayor api /
eṣa eva vidhir dṛṣṭaḥ sarvārthāpahnaveṣu ca // Nar_1.220 //
śapathā hy api devānām ṛṣīṇām api ca smṛtāḥ /
vasiṣṭhaḥ śapathaṃ śepe yātudhāne tu śaṅkitaḥ // Nar_1.221 //
saptarṣayas tathendrāya puṣkarārthe samāgatāḥ /
śepuḥ śapatham avyagrāḥ parasparaviśuddhaye // Nar_1.222 //
ayuktaṃ sāhasaṃ kṛtvā pratyāpattiṃ bhajeta yaḥ /
brūyāt svayaṃ vā sadasi tasyārdhavinayaḥ smṛtaḥ // Nar_1.223 //
gūhamānas tu daurātmyād yadi pāpaṃ sa jīyate /
sabhyāś cātra na tuṣyanti tīvro daṇḍaś ca pātyate // Nar_1.224 //
2.nikṣipaḥ
svadravyaṃ yatra viśrambhān nikṣipaty aviśaṅkitaḥ /
nikṣepo nāma tat proktaṃ vyavahārapadaṃ budhaiḥ // Nar_2.01 //
anyadravyavyavahitaṃ dravyam avyākṛtaṃ ca yat /
nikṣipyate paragṛhe tad aupanidhikaṃ smṛtam // Nar_2.02 //
sa punar dvividhaḥ proktaḥ sākṣimān itaras tathā /
pratidānaṃ tathaivāsya pratyayaḥ syād viparyaye // Nar_2.03 //
yācyamānas tu yo dātrā nikṣepaṃ na prayacchati /
daṇḍyaḥ sa rājñā dāpyaś ca naṣṭe dāpyaś ca tatsamam // Nar_2.04 //
yaś cārthaṃ sādhayet tena nikṣeptur ananujñayā /
tatrāpi daṇḍyaḥ sa bhavet tac ca sodayam āvahet // Nar_2.05 //
grahītuḥ saha yo 'rthena naṣṭo naṣṭaḥ sa dāyinaḥ /
daivarājakṛte tadvan na cet taj jihmakāritam // Nar_2.06 //
eṣa eva vidhir dṛṣṭo yācitānvāhitādiṣu /
śilpiṣūpanidhau nyāse pratinyāse tathaiva ca // Nar_2.07 //
pratigṛhṇāti pogaṇḍaṃ yaś ca sapradhanaṃ naraḥ /
tasyāpy eṣa bhaved dharmaḥ ṣaḍ ete vidhayaḥ samāḥ // Nar_2.08 //
3.saṃbhūyasamutthānam
vaṇikprabhṛtayo yatra karma saṃbhūya kurvate /
tat saṃbhūyasamutthānaṃ vyavahārapadaṃ smṛtam // Nar_3.01 //
phalahetor upāyena karma saṃbhūya kurvatām /
ādhārabhūtaḥ prakṣepas tenottiṣṭheyur aṃśataḥ // Nar_3.02 //
samo 'tirikto hīno vā yatrāṃśo yasya yādṛśaḥ /
kṣayavyayau tathā vṛddhis tasya tatra tathāvidhāḥ // Nar_3.03 //
bhāṇḍapiṇḍavyayoddhāra- bhārasārānvavekṣaṇam /
kuryus te 'vyabhicāreṇa samaye sve vyavasthitāḥ // Nar_3.04 //
pramādān nāśitaṃ dāpyaḥ pratiṣiddhakṛtaṃ ca yat /
asaṃdiṣṭaś ca yat kuryāt sarvaiḥ saṃbhūyakāribhiḥ // Nar_3.05 //
daivataskararājotthe vyasane samupasthite /
yas tat svaśaktyā saṃrakṣet tasyāṃśo daśamaḥ smṛtaḥ // Nar_3.06 //
ekasya cet syād vyasanaṃ dāyādo 'sya tad āpnuyāt /
anyo vāsati dāyāde śaktāś cet sarva eva vā // Nar_3.07 //
ṛtvijāṃ vyasane 'py evam anyas tat karma nistaret /
labheta dakṣiṇābhāgaṃ sa tasmāt saṃprakalpitam // Nar_3.08 //
ṛtvig yājyam aduṣṭaṃ yas tyajed anapakāriṇam /
aduṣṭaṃ va rtvijaṃ yājyo vineyau tāv ubhāv api // Nar_3.09 //
ṛtvik tu trividho dṛṣṭaḥ pūrvajuṣṭaḥ svayaṃkṛtaḥ /
yadṛcchayā ca yaḥ kuryād ārtvijyaṃ prītipūrvakam // Nar_3.10 //
kramāgateṣv eṣa dharmo vṛteṣv ṛtvikṣu ca svayam /
yādṛcchike tu saṃyājye tattyāge nāsti kilbiṣam // Nar_3.11 //
śulkasthānaṃ vaṇik prāptaḥ śulkaṃ dadyād yathopagam /
na tad vyatihared rājñāṃ balir eṣa prakalpitaḥ // Nar_3.12 //
śulkasthānaṃ pariharan na kāle krayavikrayī /
mithyoktvā ca parīmāṇaṃ dāpyo 'ṣṭaguṇam atyayam // Nar_3.13 //
kaścic cet saṃcaran deśāt preyād abhyāgato vaṇik /
rājāsya bhāṇḍaṃ tad rakṣet yāvad dāyādadarśanam // Nar_3.14 //
dāyāde 'sati bandhubhyo jñātibhyo vā tad arpayet /
tadabhāve suguptaṃ tad dhārayed daśatīḥ samāḥ // Nar_3.15 //
asvāmikam adāyādaṃ daśavarṣasthitaṃ tataḥ /
rājā tad ātmasāt kuryād evaṃ dharmo na hīyate // Nar_3.16 //
4.dattāpradānikam
dattvā dravyam asamyag yaḥ punar ādātum icchati /
dattāpradānikaṃ nāma tad vivādapadaṃ smṛtam // Nar_4.01 //
adeyam atha deyaṃ ca dattaṃ cādattam eva ca /
vyavahāreṣu vijñeyo dānamārgaś caturvidhaḥ // Nar_4.02 //
tatra hyaṣṭāv adeyāni deyam ekavidhaṃ smṛtam /
dattaṃ saptavidhaṃ vidyād adattaṃ ṣoḍaśātmakam // Nar_4.03 //
anvāhitaṃ yācitakam ādhiḥ sādhāraṇaṃ ca yat /
nikṣepaḥ putradāraṃ ca sarvasvaṃ cānvaye sati // Nar_4.04 //
āpatsv api hi kaṣṭāsu vartamānena dehinā /
adeyāny āhur ācāryā yac cānyasmai pratiśrutam // Nar_4.05 //
kuṭumbabharaṇād dravyaṃ yatkiṃcid atiricyate /
tad deyam upahṛtyānyad dadad doṣam avāpnuyāt // Nar_4.06 //
paṇyamūlyaṃ bhṛtis tuṣṭyā snehāt pratyupakārataḥ /
strīśulkānugrahārthaṃ ca dattaṃ dānavido viduḥ // Nar_4.07 //
adattaṃ tu bhayakrodha- śokavegarujānvitaiḥ /
tathotkocaparīhāsa- vyatyāsacchalayogataḥ // Nar_4.08 //
bālamūḍhāsvatantrārta- mattonmattāpavarjitam /
kartā mamāyaṃ karmeti pratilābhecchayā ca yat // Nar_4.09 //
apātre pātram ity ukte kārye cādharmasaṃhite /
yad dattaṃ syād avijñānād adattaṃ tad api smṛtam // Nar_4.10 //
gṛhṇāty adattaṃ yo lobhād yaś cādeyaṃ prayacchati /
adattādāyako daṇḍyas tathādeyasya dāyakaḥ // Nar_4.11 //
5.abhyupetyāśuśrūṣā
abhyupetya tu śuśrūṣāṃ yas tāṃ na pratipadyate /
aśuśrūṣābhyupetyaitad vivādapadam ucyate // Nar_5.01 //
śuśrūṣakaḥ pañcavidhaḥ śāstre dṛṣṭo manīṣibhiḥ /
caturvidhaḥ karmakaras teṣāṃ dāsās tripañcakāḥ // Nar_5.02 //
śiṣyāntevāsibhṛtakāś caturthas tv adhikarmakṛt /
ete karmakarāḥ proktā dāsās tu gṛhajādayaḥ // Nar_5.03 //
sāmānyam asvatantratvam eṣām āhur manīṣiṇaḥ /
jātikarmakṛtas tūkto viśeṣo vṛttir eva ca // Nar_5.04 //
karmāpi dvividhaṃ jñeyam aśubhaṃ śubham eva ca /
aśubhaṃ dāsakarmoktaṃ śubhaṃ karmakṛtāṃ smṛtam // Nar_5.05 //
gṛhadvārāśucisthāna- rathyāvaskaraśodhanam /
guhyāṅgasparśanocchiṣṭa- viṇmūtragrahaṇojjhanam // Nar_5.06 //
iṣṭataḥ svāminaś cāṅgair upasthānam athāntataḥ /
aśubhaṃ karma vijñeyaṃ śubham anyad ataḥ param // Nar_5.07 //
ā vidyāgrahaṇāc chiṣyaḥ śuśrūṣet prayato gurum /
tadvṛttir gurudāreṣu guruputre tathaiva ca // Nar_5.08 //
brahmacārī cared bhaikṣam adhaḥśāyy analaṅkṛtaḥ /
jaghanyaśāyī sarveṣāṃ pūrvotthāyī guror gṛhe // Nar_5.09 //
nāsaṃdiṣṭaḥ pratiṣṭheta tiṣṭhed vāpi guruṃ kvacit /
saṃdiṣṭaḥ karma kurvīta śaktaś ced avicārayan // Nar_5.10 //
yathākālam adhīyīta yāvan na vimanā guruḥ /
āsīno 'dho guroḥ kūrce phalake vā samāhitaḥ // Nar_5.11 //
anuśāsyaś ca guruṇā na ced anuvidhīyate /
avadhenāthavā hanyāt rajjvā veṇudalena vā // Nar_5.12 //
bhṛśaṃ na tāḍayed enaṃ nottamāṅge na vakṣasi /
anuśāsyātha viśvāsyaḥ śāsyo rājñānyathā guruḥ // Nar_5.13 //
samāvṛttaś ca gurave pradāya gurudakṣiṇām /
pratīyāt svagṛhān eṣā śiṣyavṛttir udāhṛtā // Nar_5.14 //
svaśilpam icchann āhartuṃ bāndhavānām anujñayā /
ācāryasya vased ante kālaṃ kṛtvā suniścitam // Nar_5.15 //
ācāryaḥ śikṣayed enaṃ svagṛhād dattabhojanam /
na cānyat kārayet karma putravac cainam ācaret // Nar_5.16 //
śikṣayantam aduṣṭaṃ ca yas tv ācāryaṃ parityajet /
balād vāsayitavyaḥ syād vadhabandhau ca so 'rhati // Nar_5.17 //
śikṣito 'pi kṛtaṃ kālam antevāsī samāpnuyāt /
tatra karma ca yat kuryād ācāryasyaiva tatphalam // Nar_5.18 //
gṛhītaśilpaḥ samaye kṛtvācāryaṃ pradakṣiṇam /
śaktitaś cānumānyainam antevāsī nivartayet // Nar_5.19 //
bhṛtakas trividho jñeya uttamo madhyamo 'dhamaḥ /
śaktibhaktyanurūpā syād eṣāṃ karmāśrayā bhṛtiḥ // Nar_5.20 //
uttamas tv āyudhīyo 'tra madhyamas tu kṛṣīvalaḥ /
adhamo bhāravāhaḥ syād ity evaṃ trividho bhṛtaḥ // Nar_5.21 //
artheṣv adhikṛto yaḥ syāt kuṭumbasya tathopari /
so 'dhikarmakaro jñeyaḥ sa ca kauṭumbikaḥ smṛtaḥ // Nar_5.22 //
śubhakarmakarās tv ete catvāraḥ samudāhṛtāḥ /
jaghanyakarmabhājas tu śeṣā dāsās tripañcakāḥ // Nar_5.23 //
gṛhajātas tathā krīto labdho dāyād upāgataḥ /
anākālabhṛtas tadvad ādhattaḥ svāminā ca yaḥ // Nar_5.24 //
mokṣito mahataś carṇāt prāpto yuddhāt paṇe jitaḥ /
tavāham ity upagataḥ pravrajyāvasitaḥ kṛtaḥ // Nar_5.25 //
bhaktadāsaś ca vijñeyas tathaiva vaḍavābhṛtaḥ /
vikretā cātmanaḥ śāstre dāsāḥ pañcadaśā smṛtāḥ // Nar_5.26 //
tatra pūrvaś caturvargo dāsatvān na vimucyate /
prasādād svāmino 'nyatra dāsyam eṣāṃ kramāgatam // Nar_5.27 //
yaś caiṣāṃ svāminaṃ kaścin mokṣayet prāṇasaṃśayāt /
dāsatvāt sa vimucyeta putrabhāgaṃ labheta ca // Nar_5.28 //
anākālabhṛto dāsyān mucyate goyugaṃ dadat /
saṃbhakṣitaṃ yad durbhikṣe na tac chudhyeta karmaṇā // Nar_5.29 //
ādhatto 'pi dhanaṃ dattvā svāmī yady enam uddharet /
athopagamayed enaṃ sa vikrītād anantaraḥ // Nar_5.30 //
dattvā tu sodayam ṛṇaṃ ṛṇī dāsyāt pramucyate /
kṛtakālābhyupagamāt kṛtako 'pi vimucyate // Nar_5.31 //
tavāham ity upagato yuddhaprāptaḥ paṇe jitaḥ /
pratiśīrṣapradānena mucyate tulyakarmaṇā // Nar_5.32 //
rājña eva tu dāsaḥ syāt pravrajyāvasito naraḥ /
na tasya pratimokṣo 'sti na viśuddhiḥ kathaṃcana // Nar_5.33 //
bhaktasyopekṣaṇāt sadyo bhaktadāsaḥ pramucyate /
nigrahād vaḍavāyāś ca mucyate vaḍavābhṛtaḥ // Nar_5.34 //
vikrīṇīte ya ātmānaṃ svatantraḥ san narādhamaḥ /
sa jaghanyataras teṣāṃ naiva dāsyāt pramucyate // Nar_5.35 //
caurāpahṛtavikrītā ye ca dāsīkṛtā balāt /
rājñā mokṣayitavyās te dāsatvaṃ teṣu neṣyate // Nar_5.36 //
varṇānāṃ prātilomyena dāsatvaṃ na vidhīyate /
svadharmatyāgino 'nyatra dāravad dāsatā matā // Nar_5.37 //
tavāham iti cātmānaṃ yo 'svatantraḥ prayacchati /
na sa taṃ prāpnuyāt kāmaṃ pūrvasvāmī labheta tam // Nar_5.38 //
adhanās traya evoktā bhāryā dāsas tathā sutaḥ /
yat te samadhigacchanti yasya te tasya tad dhanam // Nar_5.39 //
svadāsam icched yaḥ kartum adāsaṃ prītamānasaḥ /
skandhād ādāya tasyāpi bhindyāt kumbhaṃ sahāmbhasā // Nar_5.40 //
akṣatābhiḥ sapuṣpābhir mūrdhany enam avākiret /
adāsa iti coktvā triḥ prāṅmukhaṃ tam athotsṛjet // Nar_5.41 //
tataḥprabhṛti vaktavyaḥ svāmyanugrahapālitaḥ /
bhojyānnaḥ pratigṛhyaś ca bhavaty abhimataś ca saḥ // Nar_5.42 //
6.vetanasyānapākarma
bhṛtānāṃ vetanasyokto dānādānavidhikramaḥ /
vetanasyānapākarma tad vivādapadaṃ smṛtam // Nar_6.1 //
bhṛtāya vetanaṃ dadyāt karmasvāmī yathākramam /
ādau madhye 'vasāne vā karmaṇo yad viniścitam // Nar_6.2 //
bhṛtāvaniścitāyāṃ tu daśabhāgaṃ samāpnuyuḥ /
lābhagobījasasyānāṃ vaṇiggopakṛṣībalāḥ // Nar_6.3 //
karmopakaraṇaṃ caiṣāṃ kriyāṃ prati yad āhṛtam /
āptabhāvena kurvīta na jihmena samācaret // Nar_6.4 //
karmākurvan pratiśrutya kāryo dattvā bhṛtiṃ balāt /
bhṛtiṃ gṛhītvākurvāṇo dviguṇāṃ bhṛtim āvahet // Nar_6.5 //
kāle 'pūrṇe tyajet karma bhṛtināśo 'sya cārhati /
svāmidoṣād apākrāman yāvat kṛtam avāpnuyāt // Nar_6.6 //
bhṛtiṣaḍbhāgam ābhāṣya pathi yugyakṛtaṃ tyajan /
adadat kārayitvā tu karmaivaṃ sodayāṃ bhṛtim // Nar_6.7 //
anayan bhāṭayitvā tu bhāṇḍavān yānavāhane /
dāpyo bhṛticaturbhāgaṃ samam ardhapathe tyajan // Nar_6.8 //
anayan vāhako 'py evaṃ bhṛtihānim avāpnuyāt /
dviguṇāṃ tu bhṛtiṃ dāpyaḥ prasthāne vighnam ācaran // Nar_6.9 //
bhāṇḍaṃ vyasanam āgacched yadi vāhakadoṣataḥ /
dāpyo yat tatra naṣṭaṃ syād daivarājakṛtād ṛte // Nar_6.10 //
gavāṃ śatād vatsatarī dhenuḥ syād dviśatād bhṛtiḥ /
prati samvatsaraṃ gope sadohaś cāṣṭame 'hani // Nar_6.11 //
upānayet gā gopāya pratyahaṃ rajanīkṣaye /
cīṛṇāḥ pītāś ca tā gopaḥ sāyāhne pratyupānayet // Nar_6.12 //
syāc ced govyasanaṃ gopo vyāyacchet tatra śaktitaḥ /
aśaktas tūrṇam āgamya svāmine tan nivedayet // Nar_6.13 //
avyāyac channavikrośan svāmine cānivedayan /
voḍhum arhati gopas tāṃ vinayaṃ cāpi rājani // Nar_6.14 //
naṣṭavinaṣṭaṃ kṛmibhiḥ śvahataṃ viṣame mṛtam /
hīnaṃ puruṣakāreṇa gopāyaiva nipātayet // Nar_6.15 //
ajāvike tathāruddhe vṛkaiḥ pāle tv anāyati /
yat prasahya vṛko hanyāt pāle tatkilbiṣam bhavet // Nar_6.16 //
tāsāṃ caivāniruddhānāṃ carantīnāṃ mitho vane /
yām utpatya vṛko hanyān na pālas tatra kilbiṣī // Nar_6.17 //
vighuṣya tu hṛtaṃ caurair na pālo dātum arhati /
yadi deśe ca kāle ca svāminaḥ svasya śaṃsati // Nar_6.18 //
etena sarvapālānāṃ vivādaḥ samudāhṛtaḥ /
mṛteṣu ca viśuddhiḥ syāt pālasyāṅkādidarśanāt // Nar_6.19 //
śulkaṃ gṛhītvā paṇyastrī necchantī dvis tad āvahet /
aprayacchaṃs tadā śulkam anubhūya pumān striyam // Nar_6.20 //
ayonau kramate yas tu bahubhir vāpi vāsayet /
śulkam aṣṭaguṇaṃ dāpyo vinayas tāvad eva ca // Nar_6.21 //
parājire gṛhaṃ kṛtvā stomaṃ dattvā vaset tu yaḥ /
sa tad gṛhītvā nirgacchet tṛṇakāṣṭheṣṭakādikam // Nar_6.22 //
stomavāhīni bhāṇḍāni pūrṇakālāny upānayet /
grahītur ābhaved bhagnaṃ naṣṭaṃ cānyatra saṃplavāt // Nar_6.23 //
7.asvāmivikrayaḥ
nikṣiptaṃ vā paradravyaṃ naṣṭaṃ labdhvāpahṛtya vā /
vikrīyate 'samakṣaṃ yad vijñeyo 'svāmivikrayaḥ // Nar_7.1 //
dravyam asvāmivikrītaṃ prāpya svāmī samāpnuyāt /
prakāśaṃ krayataḥ śuddhiḥ kretuḥ steyaṃ rahaḥ krayāt // Nar_7.2 //
asvāmyanumatād dāsād asataś ca janād rahaḥ /
hīnamūlyam avelāyāṃ krīṇaṃs taddoṣabhāg bhavet // Nar_7.3 //
na gūhetāgamaṃ kretā śuddhis tasya tadāgamāt /
viparyaye tulyadoṣaḥ steyadaṇḍaṃ ca so 'rhati // Nar_7.4 //
vikretā svāmine 'rthaṃ ca kretur mūlyaṃ ca tatkṛtam /
dadyād daṇḍaṃ tathā rājñe vidhir asvāmivikraye // Nar_7.5 //
pareṇa nihitaṃ labdhvā rājany upaharen nidhim /
rājagāmī nidhiḥ sarvaḥ sarveṣāṃ brāhmaṇād ṛte // Nar_7.6 //
brāhmaṇo 'pi nidhiṃ labdhvā kṣipraṃ rājñe nivedayet /
tena dattaṃ ca bhūñjīta stenaḥ syād anivedayan // Nar_7.7 //
svam apy arthaṃ tathā naṣṭaṃ labdhvā rājñe nivedayet /
gṛhṇīyāt tatra taṃ śuddham aśuddhaṃ syāt tato 'nyathā // Nar_7.8 //
8.krītānuśayaḥ
vikrīya paṇyaṃ mūlyena kretur yan na pradīyate /
vikrīyāsaṃpradānaṃ tad vivādapadam ucyate // Nar_8.1 //
loke 'smin dvividhaṃ dravyaṃ jaṅgamaṃ sthāvaraṃ tathā /
krayavikrayadharmeṣu sarvaṃ tat paṇyam ucyate // Nar_8.2 //
ṣaḍvidhas tasya tu budhair dānādānavidhiḥ smṛtaḥ /
gaṇimaṃ tulimaṃ meyaṃ kriyayā rūpataḥ śriyā // Nar_8.3 //
vikrīya paṇyaṃ mūlyena kretur yo na prayacchati /
sthāvarasya kṣayaṃ dāpyo jaṅgamasya kriyāphalam // Nar_8.4 //
arghaś ced apahīyeta sodayaṃ paṇyam āvahet /
sthāyinām eṣa niyamo diglābho digvicāriṇām // Nar_8.5 //
upahanyeta vā paṇyaṃ dahyetāpahriyeta vā /
vikretur eva so 'nartho vikrīyāsamprayacchataḥ // Nar_8.6 //
nirdoṣaṃ darśayitvā tu sadoṣaṃ yaḥ prayacchati /
mūlyaṃ taddviguṇaṃ dāpyo vinayaṃ tāvad eva ca // Nar_8.7 //
tathānyahaste vikrīya yo 'nyasmai saṃprayacchati /
so 'pi taddviguṇaṃ dāpyo vineyas tāvad eva ca // Nar_8.8 //
dīyamānaṃ na gṛhṇāti krītaṃ paṇyaṃ ca yaḥ krayī /
vikrīṇānas tad anyatra vikretā nāparādhnuyāt // Nar_8.9 //
dattamūlyasya paṇyasya vidhir eṣa prakīrtitaḥ /
adatte 'nyatra samayān na vikretur atikramaḥ // Nar_8.10 //
lābhārthe vaṇijāṃ sarva- paṇyeṣu krayavikrayaḥ /
sa ca lābho 'rgham āsādya mahān bhavati vā na vā // Nar_8.11 //
tasmād deśe ca kāle ca vaṇig arghaṃ parākramet /
na jihmena pravarteta śreyān evaṃ vaṇikpathaḥ // Nar_8.12 //
9.vikrīyāsaṃpradānam
krītvā mūlyena yaḥ paṇyaṃ kretā na bahu manyate /
krītvānuśaya ity etad vivādapadam ucyate // Nar_9.1 //
krītvā mūlyena yat paṇyaṃ duṣkrītaṃ manyate krayī /
vikretuḥ pratideyaṃ tat tasminn evāhny avikṣatam // Nar_9.2 //
dvitīye 'hni dadat kretā mūlyāt triṃśāṃśam āvahet /
dviguṇaṃ tat tṛtīye 'hni parataḥ kretur eva tat // Nar_9.3 //
kretā paṇyaṃ parīkṣeta prāk svayaṃ guṇadoṣataḥ /
parīkṣyābhimataṃ krītaṃ vikretur na bhavet punaḥ // Nar_9.4 //
tryahād dohyaṃ parīkṣeta pañcāhād vāhyam eva tu /
muktāvajrapravālānāṃ saptāhaṃ syāt parīkṣaṇam // Nar_9.5 //
dvipadām ardhamāsaṃ syāt puṃsāṃ taddviguṇaṃ striyāḥ /

pāṇau yaś ca nigṛhṇīyad veṇyāṃ vastrāntare 'pi vā /

viparyaye madhyamas tu prātilome pramāpaṇam // Nar_12.69 //

śiśnasyotkartanaṃ daṇḍo nānyas tatra vidhīyate // Nar_12.74 //
paśuyonyām atikrāman vineyaḥ sa damaṃ śatam /
madhyamaṃ sāhasaṃ goṣu tad evāntyāvasāyiṣu // Nar_12.75 //
agamyāgāminaḥ śāsti daṇḍo rājñā pracoditaḥ /
prāyaścittavidhāv atra prāyaścittaṃ viśodhanam // Nar_12.76 //

anutpannaprajāyās tu patiḥ preyād yadi striyāḥ /
niyuktā gurubhir gacched devaraṃ putrakāmyayā // Nar_12.79 //
sa ca tāṃ pratipadyeta tathaivā putrajanmataḥ /
putre jāte nivarteta viplavaḥ syād ato 'nyathā // Nar_12.80 //

aniyuktā tu yā nārī devarāj janayet sutam /
jārajātam arikthīyaṃ tam āhur dharmavādinaḥ // Nar_12.83 //
tathāniyukto bhāryāyāṃ yavīyāñ jyāyaso vrajet /
yavīyaso vā yo jyāyān ubhau tau gurutalpagau // Nar_12.84 //
kule tadavaśeṣe tu saṃtānārthaṃ na kāmataḥ /
niyukto gurubhir gacched bhrātṛbhāryāṃ yavīyasaḥ // Nar_12.85 //

tatas tadvacanād gacched anuśiṣya striyā saha // Nar_12.86 //
pūrvoktenaiva vidhinā snātāṃ puṃsavane śuciḥ /
sakṛd ā garbhādhānād vā kṛte garbhe snuṣaiva sā // Nar_12.87 //
ato 'nyathā vartamānaḥ pumān strī vāpi kāmataḥ /
vineyau subhṛśaṃ rājñā kilbiṣī syād anigrahāt // Nar_12.88 //

ekāntaras tu dauṣṣanto vaiśyāyāṃ brāhmaṇāt sutaḥ /

prātilomyena varṇānāṃ tadvad ete 'py anantarāḥ // Nar_12.113 //

māgadhāyogavau tadvad dvī putrau vaiśyaśūdrayoḥ // Nar_12.114 //

dvyantaraḥ prātilomyena pāpiṣṭhaḥ sati saṃkare /

utkrośatāṃ janānāṃ ca hriyamāṇe dhane 'pi ca /
śrutvā ye nābhidhāvanti te 'pi taddoṣabhāginaḥ // Nar_14.19 //
sāhaseṣu ya evoktas triṣu daṇḍo manīṣibhiḥ /
sa eva daṇḍaḥ steye 'pi dravyeṣu triṣv anukramāt // Nar_14.20 //
gavādiṣu praṇaṣṭeṣu dravyeṣv apahṛteṣu vā /
padenānveṣaṇaṃ kuryur ā mūlāt tadvido janāḥ // Nar_14.21 //
grāme vraje vivīte vā yatra saṃnipatet padam /
voḍhavyaṃ tad bhavet tena na cet so 'nyatra tan nayet // Nar_14.22 //
pade pramūḍhe bhagne vā viṣamatvāj janāntike /
yas tv āsannataro grāmo vrajo vā tatra pātayet // Nar_14.23 //
same 'dhvani dvayor yatra tena prāyo 'śucir janaḥ /
pūrvāpadānair dṛṣṭo vā saṃsṛṣṭo vā durātmabhiḥ // Nar_14.24 //
grāmeṣv anveṣaṇaṃ kuryuś caṇḍālavadhakādayaḥ /
rātrisaṃcāriṇo ye ca bahiḥ kuryur bahiścarāḥ // Nar_14.25 //
steneṣv alabhyamāneṣu rājā dadyāt svakād dhanāt /
upekṣamāṇo hy enasvī dharmād arthāc ca hīyate // Nar_14.26 //
deśajātikulādīnām ākrośanyaṅgasaṃhitam /
yad vacaḥ pratikūlārthaṃ vākpāruṣyaṃ tad ucyate // Nar_1516.1 //
niṣṭhurāślīlatīvratvāt tad api trividhaṃ smṛtam /
gauravānukramād asya daṇḍo 'py atra kramād guruḥ // Nar_1516.2 //
sākṣepaṃ niṣṭhuraṃ jñeyam aślīlaṃ nyaṅgasaṃyutam /
pātanīyair upakrośais tīvram āhur manīṣiṇaḥ // Nar_1516.3 //
paragātreṣv abhidroho hastapādāyudhādibhiḥ /
bhasmādibhiś copaghāto daṇḍapāruṣyam ucyate // Nar_1516.4 //
tasyāpi dṛṣṭaṃ traividhyaṃ mṛdumadhyottamaṃ kramāt /
avagūraṇaniḥsaṅga- pātanakṣatadarśanaiḥ // Nar_1516.5 //
hīnamadhyottamānāṃ tu dravyāṇām samatikramāt /
trīṇy eva sāhasāny āhus tatra kaṇṭakaśodhanam // Nar_1516.6 //
vidhiḥ pañcavidhas tūkta etayor ubhayor api /
viśuddhir daṇḍabhāktvaṃ ca tatra saṃbadhyate yathā // Nar_1516.7 //
pāruṣye sati saṃrambhād utpanne kṣubdhayor dvayoḥ /
sa manyate yaḥ kṣamate daṇḍabhāg yo 'tivartate // Nar_1516.8 //
pāruṣyadoṣāvṛtayor yugapat saṃpravṛttayoḥ /
viśeṣaś cen na dṛśyeta vinayaḥ syāt samas tayoḥ // Nar_1516.9 //
pūrvam ākṣārayed yas tu niyataṃ syāt sa doṣabhāk /
paścād yaḥ so 'py asatkārī pūrve tu vinayo guruḥ // Nar_1516.10 //
dvayor āpannayos tulyam anubadhnāti yaḥ punaḥ /
sa tayor daṇḍam āpnoti pūrvo vā yadi vetaraḥ // Nar_1516.11 //
śvapākapaṇḍacaṇḍāla- vyaṅgeṣu vadhavṛttiṣu /
hastipavrātyadāreṣu gurvācāryāṅganāsu ca // Nar_1516.12 //
maryādātikrame sadyo ghāta evānuśāsanam /
na ca taddaṇḍapāruṣye doṣam āhur manīṣiṇaḥ // Nar_1516.13 //
yam eva hy ativarterann ete santaṃ janaṃ nṛṣu /
sa eva vinayaṃ kuryān na tadvinayabhāṅ nṛpaḥ // Nar_1516.14 //
malā hy ete manuṣyeṣu dhanam eṣāṃ malātmakam /
api tān ghātayed rājā nārthadaṇḍena daṇḍayet // Nar_1516.15 //
śataṃ brāhmaṇam ākruśya kṣatriyo daṇḍam arhati /
vaiśyo 'dhyardhaṃ śataṃ dve vā śūdras tu vadham arhati // Nar_1516.16 //
vipraḥ pañcāśataṃ daṇḍyaḥ kṣatriyasyābhiśaṃsane /
vaiśye syād ardhapañcāśac chūdre dvādaśako damaḥ // Nar_1516.17 //
samavarṇadvijātīnāṃ dvādaśaiva vyatikrame /
vādeṣv avacanīyeṣu tad eva dviguṇaṃ bhavet // Nar_1516.18 //
kāṇam apy athavā khañjam anyaṃ vāpi tathāvidham /
tathyenāpi bruvan dāpyo rājñā kārṣāpaṇāvaram // Nar_1516.19 //
na kilbiṣeṇāpavadec chāstrataḥ kṛtapāvanam /
na rājñā dhṛtadaṇḍaṃ ca daṇḍabhāk tadvyatikramāt // Nar_1516.20 //
loke 'smin dvāv avaktavyāv adaṇḍyau ca prakīrtitau /
brāhmaṇaś caiva rājā ca tau hīdaṃ bibhṛto jagat // Nar_1516.21 //
patitaṃ patitety uktvā cauraṃ caureti vā punaḥ /
vacanāt tulyadoṣaḥ syān mithyā dvir doṣatāṃ vrajet // Nar_1516.22 //
nāmajātigrahaṃ teṣām abhidroheṇa kurvataḥ /
nikheyo 'yomayaḥ śaṅkuḥ śūdrasyāṣṭādaśāṅgulaḥ // Nar_1516.23 //
dharmāpadeśaṃ darpeṇa dvijānām asya kurvataḥ /
taptam āsecayet tailaṃ vaktre śrotre ca pārthivaḥ // Nar_1516.24 //
yenāṅgenāvaro varṇo brāhmaṇasyāparādhnuyāt /
tad aṅgaṃ tasya chettavyam evaṃ śuddhim avāpnuyāt // Nar_1516.25 //
sahāsanam abhiprepsur utkṛṣṭasyāvakṛṣṭajaḥ /
kaṭyāṃ kṛṭāṅko nirvāsyaḥ sphigdeśaṃ vāsya kartayet // Nar_1516.26 //
avaniṣṭhīvato darpād dvāv oṣṭhau chedayen nṛpaḥ /
avamūtrayataḥ śiśnam avaśardhayato gudam // Nar_1516.27 //
keśeṣu gṛhṇato hastau chedayed avicārayan /
pādayor nāsikāyāṃ ca grīvāyāṃ vṛṣaṇeṣu ca // Nar_1516.28 //

sabhikaḥ kārayed dyūtaṃ deyaṃ dadyāc ca tatkṛtam /
daśakaṃ tu śataṃ vṛddhis tasya syād dyūtakāritā // Nar_17.2 //
dvirabhyastāḥ patanty akṣā glahe yasyākṣadevinaḥ /
jayaṃ tasyāparasyāhuḥ kitavasya parājayam // Nar_17.3 //
kitaveṣv eva tiṣṭheyuḥ kitavāḥ saṃśayaṃ prati /
ta eva tasya draṣṭāraḥ syus ta eva ca sākṣiṇaḥ // Nar_17.4 //
aśuddhaḥ kitavo nānyad āśrayed dyūtamaṇḍalam /
pratihanyān na sabhikaṃ dāpayet tat svam iṣṭataḥ // Nar_17.5 //
kūṭākṣadevinaḥ pāpān nirbhajed dyūtamaṇḍalāt /
kaṇṭhe 'kṣamālām āsajya sa hy eṣāṃ vinayaḥ smṛtaḥ // Nar_17.6 //
prakīrṇake punar jñeyā vyavahārā nṛpāśrayāḥ /
rājñām ājñāpratīghātas tatkarmakaraṇaṃ tathā // Nar_18.1 //
purapradānaṃ saṃbhedaḥ prakṛtīnāṃ tathaiva ca /
pāṣaṇḍanaigamaśreṇī- gaṇadharmaviparyayāḥ // Nar_18.2 //
pitṛputravivādaś ca prāyaścittavyatikramaḥ /
pratigrahavilopaś ca kopa āśramiṇām api // Nar_18.3 //
varṇasaṃkaradoṣaś ca tadvṛttiniyamas tathā /
na dṛṣṭaṃ yac ca pūrveṣu tat sarvaṃ syāt prakīrṇake // Nar_18.4 //

yo yo varṇo 'vahīyeta yo vodrekam anuvrajet /
taṃ taṃ dṛṣṭvā svato mārgāt pracyutaṃ sthāpayet pathi // Nar_18.6 //
aśāstrokteṣu cānyeṣu pāpayukteṣu karmasu /

śrutismṛtiviruddhaṃ ca janānām ahitaṃ ca yat /
na tat pravartayed rājā pravṛttaṃ ca nivartayet // Nar_18.8 //
nyāyāpetaṃ yad anyena rājñājñānakṛtaṃ ca yat /
tad apy anyāyavihitaṃ punar nyāye niveśayet // Nar_18.9 //
rājñā pravartitān dharmān yo naro nānupālayet /
daṇḍyaḥ sa pāpo vadhyaś ca lopayan rājaśāsanam // Nar_18.10 //
āyudhāny āyudhīyānāṃ vāhyādīn vāhyajīvinām /

sarvasvaharaṇe 'py etān na rājā hartum arhati // Nar_18.12 //

dīptimatvāc chucitvāc ca yadi na syāt pathaś cyutaḥ // Nar_18.13 //

kuryāt patho vyapetānāṃ vinaśyeyur imāḥ prajāḥ // Nar_18.14 //
brāhmaṇyaṃ brāhmaṇo jahyāt kṣatriyaḥ kṣātram utsṛjet /

satām anugraho nityam asatāṃ nigrahas tathā /
eṣa dharmaḥ smṛto rājñām arthaś cāmitrapīḍanāt // Nar_18.17 //
na lipyate yathā vahnir dahañ chaśvad imāḥ prajāḥ /
na lipyate tathā rājā daṇḍaṃ daṇḍyeṣu pātayan // Nar_18.18 //
ājñā tejaḥ pārthivānāṃ sā ca vāci pratiṣṭhitā /
te yad brūyur asat sad vā sa dharmo vyavahāriṇām // Nar_18.19 //
rājā nāma caraty eṣa bhūmau sākṣāt sahasradṛk /

nirguṇo 'pi yathā strīṇāṃ pūjya eva patiḥ sadā /
prajānāṃ viguṇo 'py evaṃ pūjya eva narādhipaḥ // Nar_18.22 //
tapaḥkrītāḥ prajā rājñā prabhur āsāṃ tato nṛpaḥ /
tatas tadvacasi stheyaṃ vārtā cāsāṃ tadāśrayā // Nar_18.23 //

agner indrasya somasya yamasya dhanadasya ca // Nar_18.24 //
kāraṇād animittaṃ vā yadā krodhavaśaṃ gataḥ /
prajā dahati bhūpālas tadāgnir abhidhīyate // Nar_18.25 //

abhiyāti parān rājā tadendraḥ sa udāhṛtaḥ // Nar_18.26 //

prajānāṃ darśanaṃ yāti soma ity ucyate tadā // Nar_18.27 //

tasmāt taṃ nāvajānīyān nākrośen na viśeṣayet /
ājñāyāṃ cāsya tiṣṭheta mṛtyuḥ syāt tadvyatikramāt // Nar_18.30 //
tasya vṛttiḥ prajārakṣā vṛddhaprājñopasevanam /

brāhmaṇān upaseveta nityaṃ rājā samāhitaḥ /
saṃyuktaṃ brāhmaṇaiḥ kṣatraṃ mūlaṃ lokābhirakṣaṇe // Nar_18.32 //

gocare yasya muṣyeta tena caurāḥ prayatnataḥ /
mṛgyā dāpyo 'nyathā moṣaṃ padaṃ yadi na nirgatam // Nar_19.23 //
nirgate tu pade tasmin naṣṭe 'nyatra nipātite /
sāmantān mārgapālāṃś ca dikpālāṃś caiva dāpayet // Nar_19.24 //
gṛhe vai muṣite rājā cauragrāhāṃs tu dāpayet /
ārakṣakān rāṣṭrikāṃś ca yadi cauro na labhyate // Nar_19.25 //
yadi vā dāpyamānānāṃ tasmin moṣe tu saṃśayaḥ /
muṣitaḥ śapathaṃ śāpyo moṣe vaiśodhyakāraṇāt // Nar_19.26 //
acaure dāpite moṣaṃ cauryavaiśodhyakāraṇāt /
caure labdhe labheyus te dviguṇaṃ pratipāditāḥ // Nar_19.27 //
caurahṛtaṃ prayatnena sarūpaṃ pratipādayet /
tadabhāve tu mūlyaṃ syād daṇḍaṃ dāpyaś ca tatsamam // Nar_19.28 //
kāṣṭhakāṇḍatṛṇādīnāṃ mṛnmayānāṃ tathaiva ca /
veṇuvaiṇavabhāṇḍānāṃ vetrasnāyvasthicarmaṇām // Nar_19.29 //
śākaharitamūlānāṃ haraṇe phalapuṣpayoḥ /
gorasekṣuvikārāṇāṃ tathā lavaṇatailayoḥ // Nar_19.30 //
pakvānnānāṃ kṛtānnānāṃ madyānām āmiṣasya ca /
sarveṣām alpamūlyānāṃ mūlyāt pañcaguṇo damaḥ // Nar_19.31 //
tulādharimameyānāṃ gaṇimānāṃ ca sarvaśaḥ /
ebhyas tūtkṛṣṭamūlyānāṃ mūlyād daśaguṇo damaḥ // Nar_19.32 //
dhānyaṃ daśabhyaḥ kumbhebhyo harato 'bhyadhikaṃ vadhaḥ /
nyūnaṃ tv ekādaśaguṇaṃ daṇḍaṃ dāpyo 'bravīn manuḥ // Nar_19.33 //
suvarṇarajatādīnām uttamānāṃ ca vāsasām /
ratnānāṃ caiva mukhyānāṃ śatād abhyadhikaṃ vadhaḥ // Nar_19.34 //
puruṣaṃ harataḥ pātyo daṇḍa uttamasāhasaḥ /
sarvasvaṃ strīṃ tu harataḥ kanyāṃ tu harato vadhaḥ // Nar_19.35 //
mahāpaśūn stenayato daṇḍa uttamasāhasaḥ /
madhyamo madhyamapaśuṃ pūrvaḥ kṣudrapaśuṃ haran // Nar_19.36 //
caturviṃśāvaraḥ pūrvaḥ paraḥ ṣaṇṇavatir bhavet /
śatāni pañca tu paro madhyamo dviśatāvaraḥ // Nar_19.37 //
sahasraṃ tūttamo jñeyaḥ paraḥ pañcaśatāvaraḥ /
trividhaḥ sāhaseṣv eva daṇḍaḥ proktaḥ svayaṃbhuvā // Nar_19.38 //
prathame granthibhedānām aṅgulyaṅguṣṭhayor vadhaḥ /
dvitīye caiva taccheṣaṃ daṇḍaḥ pūrvaś ca sāhasaḥ // Nar_19.39 //
goṣu brāhmaṇasaṃsthāsu sthūrāyāś chedanaṃ bhavet /
dāsīṃ tu harato nityam ardhapādavikartanam // Nar_19.40 //
yena yena viśeṣeṇa stenāṅgena viceṣṭate /
tat tad evāsya chettavyaṃ tan manor anuśāsanam // Nar_19.41 //
garīyasi garīyāṃsam agarīyasi vā punaḥ /
stene nipātayed daṇḍaṃ na yathā prathame tathā // Nar_19.42 //
daśa sthānāni daṇḍasya manuḥ svāyaṃbhuvo 'bravīt /
triṣu varṇeṣu yāni syur brāhmaṇo rakṣitaḥ sadā // Nar_19.43 //
upastham udaraṃ jihvā hastau pādau ca pañcamam /
cakṣur nāsā ca karṇau ca dhanaṃ dehas tathaiva ca // Nar_19.44 //
aparādhaṃ parijñāya deśakālau ca tattvataḥ /
sārānubandhāv ālokya daṇḍān etān prakalpayet // Nar_19.45 //
na mitrakāraṇād rājñā vipulād vā dhanāgamāt /
utsraṣṭavyaḥ sāhasikas tyaktātmā manur abravīt // Nar_19.46 //
yāvān avadhyasya vadhe tāvān vadhyasya mokṣaṇe /
bhavaty adharmo nṛpater dharmas tu viniyacchataḥ // Nar_19.47 //
na jātu brāhmaṇaṃ hanyāt sarvapāpeṣv api sthitam /
nirvāsaṃ kārayet kāmam iti dharmo vyavasthitaḥ // Nar_19.48 //
sarvasvaṃ vā hared rājā caturthaṃ vāvaśeṣayet /
bhṛtyebhyo 'nusmaran dharmaṃ prājāpatyam iti sthitiḥ // Nar_19.49 //
brāhmaṇasyāparādhe tu catuḥsv aṅko vidhīyate /
gurutalpe surāpāne steye brāhmaṇahiṃsane // Nar_19.50 //
gurutalpe bhagaḥ kāryaḥ surāpāne dhvajaḥ smṛtaḥ /
steye tu śvapadaṃ kṛtvā śikhipittena kūṭayet // Nar_19.51 //
viśirāḥ puruṣaḥ kāryo lalāṭe bhrūṇaghātinaḥ /
asaṃbhāṣyaś ca kartavyas tan manor anuśāsanam // Nar_19.52 //
rājā stenena gantavyo muktakeśena dhāvatā /
ācakṣāṇena tatsteyam evaṃ kartāsmi śādhi mām // Nar_19.53 //
anenā bhavati stenaḥ svakarmapratipādanāt /
rājānaṃ tat spṛśed ena utsṛjantaṃ sakilbiṣam // Nar_19.54 //
rājabhir dhṛtadaṇḍās tu kṛtvā pāpāni mānavāḥ /
nirmalāḥ svargam āyānti santaḥ sukṛtino yathā // Nar_19.55 //
śāsanād vā vimokṣād vā steno mucyate kilbiṣāt /
aśāsanāt tu tad rājā stenasyāpnoti kilbiṣam // Nar_19.56 //
gurur ātmavatāṃ śāstā śāstā rājā durātmanām /
atha pracchannapāpānāṃ śāstā vaivasvato yamaḥ // Nar_19.57 //
aṣṭāpādyaṃ tu śūdrasya steye bhavati kilbiṣam /
dvir aṣṭāpādyaṃ vaiśyasya dvātriṃśat kṣatriyasya tu // Nar_19.58 //
brāhmaṇasya catuḥṣaṣṭīty evaṃ svāyaṃbhuvo 'bravīt /
tatrāpi ca viśeṣeṇa vidvatsv abhyadhikaṃ bhavet // Nar_19.59 //
śārīraś cārthadaṇḍaś ca daṇḍas tu dvividhaḥ smṛtaḥ /
śārīrā daśadhā proktā arthadaṇḍās tv anekadhā // Nar_19.60 //
kākaṇyādis tv arthadaṇḍaḥ sarvasvāntas tathaiva ca /
śārīras tv avarodhādir jīvitāntas tathaiva ca // Nar_19.61 //
kākaṇyādis tu yo daṇḍaḥ sa tu māṣāparaḥ smṛtaḥ /
māṣāvarādyo yaḥ proktaḥ kārṣāpaṇaparas tu saḥ // Nar_19.62 //
kārṣāpaṇāparādyas tu catuḥkārṣāpaṇaḥ paraḥ /
dvyavaro 'ṣṭāparaś cānyas tryavaro dvādaśottaraḥ // Nar_19.63 //
kārṣāpaṇādyā ye proktāḥ sarve te syuś caturguṇāḥ /
evam anye tu vijñeyāḥ prāk ca te pūrvasāhasāt // Nar_19.64 //
kārṣāpaṇo dakṣiṇasyāṃ diśi raupyaḥ pravartate /
paṇair nibaddhaḥ pūrvasyāṃ ṣoḍaśaiva paṇāḥ sa tu // Nar_19.65 //
māṣo viṃśatibhāgas tu jñeyaḥ kārṣāpaṇasya tu /
kākaṇī tu caturbhāgo māṣasya ca paṇasya ca // Nar_19.66 //
pāñcanadyāḥ pradeśe tu saṃjñā yā vyāvahārikī /
kārṣāpaṇapramāṇaṃ tu nibaddham iha vai tayā // Nar_19.67 //
kārṣāpaṇo 'ṇḍikā jñeyāś catasras tās tu dhānakaḥ /
taddvādaśa suvarṇasya dīnāraś citrakaḥ smṛtaḥ // Nar_19.68 //
vārttāṃ trayīṃ cāpy atha daṇḍanītim rājānuvartet saṃtatāpramattaḥ /
hanyād upāyair nipuṇair gṛhītān pure ca rāṣṭre nigṛhṇīyāt pāpān // Nar_19.69 //
yadā sākṣī na vidyate vivāde vadatāṃ nṛṇām /
tadā divyaiḥ parīkṣeta śapathaiś ca pṛthagvidhaiḥ // Nar_20.1 //
satyaṃ vāhanaśastrāṇi gobījarajatāni ca /
devatāpitṛpādāś ca dattāni sukṛtāni ca // Nar_20.2 //
mahāparādhe divyāni dāpayet tu mahīpatiḥ /
alpeṣu ca naraḥ śreṣṭhaḥ śapathaiḥ śāpayen naram // Nar_20.3 //
ete hi śapathāḥ proktāḥ sukarās svalpasaṃśaye /
sāhaseṣv abhiśāpe ca vidhir divyaḥ prakīrtitaḥ // Nar_20.4 //
saṃdigdhe 'rthe 'bhiyuktānāṃ pracchanneṣu viśeṣataḥ /
divyaḥ pañcavidho jñeya ity āha bhagavān manuḥ // Nar_20.5 //
dhaṭo 'gnir udakaṃ caiva viṣaṃ kośaś ca pañcamaḥ /
uktāny etāni divyāni dūṣitānāṃ viśodhane // Nar_20.6 //
saṃdigdheṣv abhiyuktānāṃ viśuddhyarthaṃ mahātmanā /
nāradena punaḥ proktāḥ satyānṛtavibhāvanāḥ /
vādino 'numatenainaṃ kārayen nānyathā budhaḥ // Nar_20.7 //
[dhaṭaḥ]
caturhastau tulāpādāv ucchrayeṇa prakīrtitau /
ṣaḍḍhastaṃ tu tayor dṛṣṭaṃ pramāṇaṃ parimāṇataḥ // Nar_20.8 //
pādayor antaraṃ hastaṃ bhaved adhyardham eva ca /
śikyadvayaṃ samāsajya dhaṭe karkaṭake dṛḍhe // Nar_20.9 //
tulayitvā naraṃ pūrvaṃ cihnaṃ kuryād dhaṭasya tu /
kakṣāsthānena taṃ tulyam avatārya tato dhaṭāt // Nar_20.10 //
samayaiḥ parigṛhyainaṃ punar āropayen naraḥ /
tasminn evaṃ kṛte sā cet kakṣe sthāpya suniścalā // Nar_20.11 //
tulito yadi vardheta śuddhaḥ syān nātre saṃśayaḥ /
samo vā hīyamāno vā na viśuddho bhaven naraḥ // Nar_20.12 //
dharmaparyāyavacanair dhaṭa ity abhidhīyase /
tvaṃ vetsi sarvabhūtānāṃ pāpāni sukṛtāni ca /
tvam eva dhaṭa jānīṣe na vidur yāni mānuṣāḥ // Nar_20.13 //
vyavahārābhiśasto 'yaṃ mānuṣas tulyate tathā /
tad eva saṃśayāpannaṃ dharmatas trātum arhasi // Nar_20.14 //
ata ūrdhvaṃ pravakṣyāmi lohasya vidhim uttamam /
dvātriṃśadaṅgulāni tu maṇḍalān maṇḍalāntaram // Nar_20.15 //
aṣṭābhir maṇḍalair evam aṅgulānāṃ śatadvayam /
caturviṃśat samākhyātaṃ saṃkhyātattvārthadarśibhiḥ // Nar_20.16 //
kalpitair maṇḍalair evam uṣitasya śucer api /
saptāśvatthasya pattrāṇi sūtreṇāveṣṭya hastayoḥ // Nar_20.17 //
vidadhyāt taptalohasya pañcāśatpalam saṃmitam /
hastābhyāṃ piṇḍam ādāya śanaiḥ saptapadaṃ vrajet // Nar_20.18 //
na maṇḍalam atikrāmen nāpy arvāk pādayet padam /
na ca pātayetāprāptaḥ yāvadbhūmir prakalpitā // Nar_20.19 //
tīrtvānena vidhānena maṇḍalāni samāhitaḥ /
adagdhaḥ sarvato yas tu sa viśuddho bhaven naraḥ // Nar_20.20 //
bhayād vā pātayate yas tv adagdho yo vibhāvyate /
punas taṃ hārayel lohaṃ sthitir eṣā purātanī /
anena vidhinā kāryo hutāśasamayaḥ smṛtaḥ // Nar_20.21 //
tvam agne sarvabhūtānām antaścarasi sākṣivat /
sukṛtaṃ duḥkṛtaṃ loken(a)
ajñātaṃ vidyate tvayā // Nar_20.22 //
pracchannāni manuṣyāṇāṃ pāpāni sukṛtāni ca /
yathāvad eva jānīṣe na vidur yāni mānuṣāḥ // Nar_20.23 //
vyavahārābhiśasto 'yaṃ puruṣaḥ śuddhim icchati /
tad enaṃ saṃśayāpannaṃ dharmatas trātum arhasi // Nar_20.24 //
ataḥ paraṃ pravakṣyāmi toyasya vidhim uttamam /
nātikrūreṇa dhanuṣā prerayet sāyakatrayam // Nar_20.25 //
madhyamas tu śaro grāhyaḥ puruṣeṇa yavīyasā /
pratyānītasya tasyātha sa viśuddho bhaven naraḥ // Nar_20.26 //
anyathā na viśuddhaḥ syād ekāṅgam api darśayet /
sthānād anyatra vā gacched yasmin pūrvaṃ niveṣitaḥ // Nar_20.27 //
striyas tu na balāt kāryā na pumān api durbalaḥ /
bhīrutvād yoṣito mṛtyuḥ kṛśasyāpi balāt kuryāt /
sahasā prāpnuyāt sarvāṃs tasmād etān na majjayet // Nar_20.28 //
toyamadhye manuṣyasya gṛhītvorū susaṃyataḥ // Nar_20.29 //
satyānṛtavibhāgasya toyāgnī spaṣṭakṛttamau /
yataś cāgnir abhūd asmāt tatas toyaṃ viśiṣyate // Nar_20.30 //
kriyate dharmatattvajñair dūṣitānāṃ viśodhanam /
tasmāt satyena bhagavañ jaleśa trātum arhasi // Nar_20.31 //
ataḥ paraṃ pravakṣyāmi viṣasya vidhim uttamam /
tulayitvā viṣaṃ pūrvaṃ deyam etad dhimāgame // Nar_20.32 //
na pūrvāhṇe na madhyāhne na saṃdhyāyāṃ tu dharmavit /
śaradgrīṣmavasanteṣu varṣāsu ca na dāpayet // Nar_20.33 //
bhagnaṃ ca dāritaṃ caiva dhūpitaṃ miśritaṃ tathā /
kālakūṭam alaṃbuṃ ca viṣaṃ yatnena varjayet // Nar_20.34 //
śārṅgahaimavataṃ śastaṃ gandhavarṇarasānvitam /
mahādoṣavate deyaṃ rājñā tattvabubhutsayā // Nar_20.35 //
na bālāturavṛddheṣu naiva svalpāparādhiṣu /
viṣasya tu yavān sapta dadyāc chodye ghṛtaplutān // Nar_20.36 //
viṣasya palaṣaḍbhāgād bhāgo viṃśatimas tu yaḥ /
tam aṣṭabhāgahīnaṃ tu śodhye dadyād ghṛtaplutam // Nar_20.37 //
yathoktena vidhānena viprān sprṣṭvānumoditaḥ /
sopavāsaś ca khādeta devabrāhmaṇasaṃnidhau // Nar_20.38 //
viṣaṃ vegaklamāpetaṃ sukhena yadi jīryate /
viśuddham iti taṃ jñātvā rājā satkṛtya mokṣayet // Nar_20.39 //
tvaṃ viṣa brahmaṇaḥ putraḥ satyadharmaratau sthitaḥ /
śodhayainaṃ naraṃ pāpāt satyenāsyāmṛtībhava // Nar_20.40 //
ataḥ paraṃ pravakṣyāmi kośasya vidhim uttamam // Nar_20.41 //
pūrvāhṇe sopavāsasya snātasyārdrapaṭasya ca /
saśūkasyāvyasaninaḥ kośapānaṃ vidhīyate // Nar_20.42 //
yadbhaktaḥ so 'bhiyuktaḥ syāt taddaivatyaṃ tu pāyayet /
saptāhād yasya dṛśyate dvisaptāhena vā punaḥ /
pratyātmikaṃ tu yatkiṃcit saiva tasya vibhāvanā // Nar_20.43 //
dvisaptāhāt paraṃ yasya mahad vā vaikṛtaṃ bhavet /
nābhiyojyaḥ sa viduṣāṃ kṛtakālavyatikramāt // Nar_20.44 //
mahāparādhe nirdharme kṛtaghne klībakutsite /
nāstikavrātyadāseṣu kośapānaṃ vivarjayet // Nar_20.45 //
yathoktena prakāreṇa pañca divyāni dharmavit /
dadyād rājābhiyuktānāṃ pretya ceha ca nandati // Nar_20.46 //
na viṣaṃ brāhmaṇe dadyān na lohaṃ kṣatriyo haret /
na nimajjyāpsu vaiśyaś ca śūdraḥ kośaṃ na pāyayet // Nar_20.47 //
varṣāsu na viṣaṃ dadyāt hemante nāpsu majjayet /
na lohaṃ hārayed grīṣme na kośaṃ pāyayen niśi // Nar_20.48 //
nāradīyadharmaśāstraḥ samāptaḥ.
