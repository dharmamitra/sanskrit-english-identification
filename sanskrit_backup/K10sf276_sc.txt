Candrasūtra
Wal_1 evaṁ mayā śrutam / ekasamayaṁ bhagavāṁ caṁpāyāṁ viharati gargarapuṣkariṇyās tīre /
Wal_2 tena khalu samayena rāhuṇā asurendreṇa sarvaṁ candramaṇḍalam āvṛtaṁ /
Wal_3 atha yā devatā tasmiṁ candramaṇḍale sthitā bhītā trastā saṁvignā hṛṣṭaromakūpā yena
bhagavāṁs tenopajagāma / upetya bhagavatpādau śirasā vanditvaikānte ’sthāt / ekāntasthitā
bhagavaṁtaṁ gāthayā babhāṣe /
Wal_4 namas te buddha vīrāstu vipramuktāya savataḥ /
saṁbādhapratipanno ’smi tasya me śaraṇaṁ bhava //
Wal_5 loke sugatam arhantaṁ candramāḥ śaraṇaṁ gataḥ /
rāhoś candramasaṁ muñca buddhā lokānukaṁpakāḥ //
Wal_6 bhagavāṁ gāthayā pratyābhāṣīt /
yo hy andhakāre nabhasi
prabhākaro vairocano … /
rāho śaśāṅkaṁ grasa māntarīkṣe
prajāpradīpaṁ tvaritaṁ vimuñca //
Wal_7 … / saṁsvinnagātro vyathitaḥ saṁbādhito yathā naraḥ / saṁtvaramāṇo ’sau rāhuś candramaṇḍalam utsṛṣṭaḥ //
Wal_8 tena samayena baḍir virocanapautra āsīd asuraḥ / adrākṣīd baḍir virocanapautro rāhuṇā asurendreṇa tvaritatvaritaṁ candramaṇḍalam utsṛṣṭam  #
dṛṣṭvā ca tasyāṁ velāyāṁ gāthāṁ babhāṣe // 
Wal_9 kiṁ nu saṁtvaramāṇas tvaṁ rāhuś candraṁ vimuñcasi / saṁsvinnagātro vyathitaḥ saṁbādhito yathā naraḥ //
Wal_10 rāhur avocat // saptadhā me sphalen mūrdhā jīvan na sukham āpnuyāṁ tatra buddhābhigītena muñceyaṁ śaśinaṁ na cet //
Wal_11 baḍir virocanapautra gāthayā babhāṣe //
… madarśinaḥ saṁbuddhāḥ … /
teṣāṁ gāthābhigītena rāhuś candraṁ vimuñcati //
Wal_12 candrasūtraṁ samāptam //
