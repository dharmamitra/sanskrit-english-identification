triśūklabhojina triṣkālasnāpayitā śūcivastraprāvṛtto bhūtvā japā dātavyaṃ /
ataḥ pratimāyā 'rpataḥ ātmānaṃ ca likhitaṃ paśyati /
taddaṣṭvā ca prahṛṣyati yāvat svapnane vā āryāvalokiteśvaro āgacchati /
sarvesā paripūrayati /
manaḥ śilāṃ rajataṃ vā parijapya kṣīṇāṃjayitvā tatoantarhito bhavati /
ākāśe krāmati /
asaṃmohavyūha nāma samādhiṃ prati labhate /
yadicchati tatkaroti eva sādhaka iti // //
idamavocat bhagavānāttamanā āryāvalokiteśvaro bodhisattvo mahāsattva rūteca bhikṣavarūte
ca bodhisatvāste śuddhāvāsakāyikāśca devaputrāḥ sadevamānuṣāsura gandharvaścaloko
bhagavato bhāṣitamabhyanandanniti // //

ye dharmmā hetuprabhavā hetusteṣāṃ tathāgataḥ /
hyavadatteṣāṃca yo nirodha evaṃ vādī mahāśramaṇaḥ //
